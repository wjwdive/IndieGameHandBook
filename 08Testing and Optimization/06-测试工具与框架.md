# 测试工具与框架

## 概述

专业的测试工具和框架是高效游戏测试的基石。对于独立开发者，选择合适的工具链能够在有限资源下最大化测试覆盖率和效率。本章将全面介绍游戏测试领域的各类工具和框架，从单元测试到性能分析，从自动化测试到持续集成，帮助建立完整的游戏测试工具生态系统。

**学习目标**：
- 了解游戏测试工具的分类和适用场景
- 掌握主流游戏引擎的测试框架配置和使用
- 学习性能分析和调试工具的核心功能
- 掌握自动化测试工具链的集成和配置
- 了解持续集成和测试报告工具的选择和使用

## 1、测试工具分类与选择

### 1.1 测试工具生态系统

游戏测试工具可以按功能和用途分为多个类别，每个类别解决特定的测试需求。

**测试工具分类矩阵**：
```markdown
| 类别 | 主要用途 | 代表工具 | 适用阶段 |
|------|----------|----------|----------|
| **单元测试框架** | 代码逻辑验证，函数级别测试 | Unity Test Framework, Godot GUT, NUnit, xUnit | 开发全过程 |
| **集成测试工具** | 模块交互测试，系统集成验证 | PlayMode Tests, 自定义测试运行器 | 模块开发完成 |
| **功能测试工具** | 用户功能验证，端到端测试 | Selenium, Appium, AltUnity Tester | 功能完整阶段 |
| **性能分析工具** | 性能瓶颈识别，优化效果验证 | Unity Profiler, RenderDoc, Intel GPA, NVIDIA Nsight | 性能关键阶段 |
| **内存分析工具** | 内存泄漏检测，资源使用优化 | Memory Profiler, dotMemory, Valgrind | 内存优化阶段 |
| **自动化测试框架** | 测试用例管理，自动化执行 | Jenkins, GitLab CI, GitHub Actions, TeamCity | 持续集成阶段 |
| **测试管理工具** | 测试用例管理，缺陷追踪 | TestRail, Zephyr, Jira, Trello | 测试全过程 |
| **用户测试平台** | 远程用户测试，反馈收集 | PlaytestCloud, UserTesting, BetaFamily | 用户验证阶段 |
```

### 1.2 工具选择决策框架

选择合适的测试工具需要考虑多个因素，以下决策框架帮助独立开发者做出明智选择。

**工具选择评估维度**：
1. **学习曲线**：工具的学习难度和上手时间
2. **集成复杂度**：与现有开发环境的集成难度
3. **社区支持**：文档质量、社区活跃度、问题解决资源
4. **成本考量**：许可费用、维护成本、基础设施需求
5. **扩展性**：是否支持自定义扩展和脚本开发
6. **跨平台支持**：是否支持所有目标平台测试
7. **团队适配**：是否适合团队规模和技能水平

**独立开发者工具选择优先级**：
```python
# 工具选择评分模型
from dataclasses import dataclass
from typing import List, Dict

@dataclass
class ToolEvaluationCriteria:
    """工具评估标准"""

    # 核心维度（权重）
    ease_of_use: float = 0.20      # 易用性
    cost: float = 0.25            # 成本（独立开发者重点）
    integration: float = 0.15      # 集成难度
    community: float = 0.15       # 社区支持
    features: float = 0.25        # 功能完整性

    # 详细评分项
    scoring_categories = {
        "ease_of_use": ["学习曲线", "文档质量", "UI直观性"],
        "cost": ["初始成本", "持续费用", "免费额度"],
        "integration": ["引擎支持", "CI/CD集成", "API可用性"],
        "community": ["活跃用户", "问题响应", "教程资源"],
        "features": ["核心功能", "扩展性", "平台覆盖"]
    }

@dataclass
class ToolScore:
    """工具评分结果"""
    tool_name: str
    category: str
    scores: Dict[str, float]  # 各维度得分（0-10）
    total_score: float
    weighted_score: float
    recommendation: str  # 推荐级别: 强烈推荐/推荐/考虑/不推荐

class ToolSelector:
    """工具选择器"""

    def __init__(self, criteria: ToolEvaluationCriteria):
        self.criteria = criteria

    def evaluate_tool(self, tool_name: str, category: str,
                     raw_scores: Dict[str, float]) -> ToolScore:
        """评估单个工具"""
        # 计算加权得分
        weighted_score = (
            raw_scores.get("ease_of_use", 5) * self.criteria.ease_of_use +
            raw_scores.get("cost", 5) * self.criteria.cost +
            raw_scores.get("integration", 5) * self.criteria.integration +
            raw_scores.get("community", 5) * self.criteria.community +
            raw_scores.get("features", 5) * self.criteria.features
        )

        # 计算总分（简单平均）
        total_score = sum(raw_scores.values()) / len(raw_scores)

        # 生成推荐级别
        if weighted_score >= 8.0:
            recommendation = "强烈推荐"
        elif weighted_score >= 6.0:
            recommendation = "推荐"
        elif weighted_score >= 4.0:
            recommendation = "考虑"
        else:
            recommendation = "不推荐"

        return ToolScore(
            tool_name=tool_name,
            category=category,
            scores=raw_scores,
            total_score=total_score,
            weighted_score=weighted_score,
            recommendation=recommendation
        )

    def compare_tools(self, tools_data: List[Dict]) -> List[ToolScore]:
        """比较多个工具"""
        results = []

        for tool_data in tools_data:
            score = self.evaluate_tool(
                tool_name=tool_data["name"],
                category=tool_data["category"],
                raw_scores=tool_data["scores"]
            )
            results.append(score)

        # 按加权得分排序
        results.sort(key=lambda x: x.weighted_score, reverse=True)
        return results

# 使用示例：Unity测试框架选择
def evaluate_unity_test_tools():
    """评估Unity测试工具"""
    criteria = ToolEvaluationCriteria(
```

## 2、单元测试与集成测试框架

### 2.1 Unity Test Framework详解

Unity Test Framework是Unity官方提供的测试解决方案，完全集成在Unity编辑器中。

**框架架构**：
- **Edit Mode Tests**：在编辑器模式下运行，不启动游戏
- **Play Mode Tests**：在游戏运行时执行，测试完整游戏行为
- **Test Runner窗口**：可视化测试管理和执行界面
- **Test Assemblies**：组织测试代码的程序集

**完整配置示例**：

### 2.2 Godot测试框架（GUT）配置

Godot Unit Test（GUT）是Godot引擎的测试框架，完全集成在Godot编辑器中。

**GUT配置与使用**：
```gdscript
# Godot GUT测试框架完整示例

# test_runner.gd - 测试运行器主脚本
extends Node

# 测试配置
var test_config = {
    "directories": ["res://tests/"],      # 测试脚本目录

    "junit_xml_timestamp": true           # 在XML中包含时间戳
}

# 测试分类标签
enum TestTags {
    UNIT = "unit",
    INTEGRATION = "integration",
    UI = "ui",
    PERFORMANCE = "performance",
    REGRESSION = "regression"
}

func _ready():

    var gut = $Gut
```

## 3、性能分析与调试工具

### 3.1 Unity Profiler深度使用

Unity Profiler是Unity引擎内置的性能分析工具，功能强大且完全集成。

**Profiler核心模块**：
1. **CPU Usage**：CPU性能分析，识别性能热点
2. **GPU Usage**：GPU渲染管线分析，识别图形瓶颈
3. **Memory**：内存使用分析，检测内存泄漏
4. **Rendering**：渲染统计，分析绘制调用和批处理
5. **Audio**：音频系统性能分析
6. **Physics**：物理系统性能分析
7. **Network**：网络通信性能分析（如使用）

**自动化性能分析脚本**：
```csharp
// Unity自动化性能分析工具
using UnityEngine;
using UnityEngine.Profiling;
using System.Collections;
using System.Collections.Generic;
using System.IO;
using System.Text;

public class AutomatedProfiler : MonoBehaviour
{
    [System.Serializable]
    public class ProfilerConfig
    {
        public string testName = "Performance Test";
        public float warmupTime = 5.0f;           // 预热时间（秒）
        public float captureTime = 30.0f;         // 捕获时间（秒）
        public int targetFPS = 60;               // 目标帧率
        public bool captureMemorySnapshot = true; // 是否捕获内存快照
        public bool captureDeepProfile = false;   // 是否深度分析
        public string outputPath = "ProfilerReports";
    }

    [Header("配置")]
    public ProfilerConfig config = new ProfilerConfig();

    [Header("性能阈值")]
    public float fpsWarningThreshold = 50.0f;     // FPS警告阈值
    public float fpsCriticalThreshold = 30.0f;    // FPS严重阈值
    public float memoryWarningThresholdMB = 512.0f; // 内存警告阈值（MB）
    public int drawCallWarningThreshold = 1000;   // 绘制调用警告阈值

    private bool isProfiling = false;
    private float startTime;
    private List<PerformanceSnapshot> snapshots = new List<PerformanceSnapshot>();
    private string reportFileName;

    [System.Serializable]
    public class PerformanceSnapshot
    {
        public float timestamp;
        public float fps;
        public float frameTimeMs;
        public float memoryMB;
        public int drawCalls;
        public int triangles;
        public int vertices;
        public float cpuMainThreadMs;
        public float cpuRenderThreadMs;
        public float gpuTimeMs;
    }

    void Start()
    {
        // 创建输出目录
        Directory.CreateDirectory(config.outputPath);

        // 生成报告文件名
        reportFileName = $"{config.testName}_{System.DateTime.Now:yyyyMMdd_HHmmss}";
    }

    public void StartProfiling()
    {
        if (isProfiling) return;

        Debug.Log($"开始性能分析: {config.testName}");
        StartCoroutine(ProfilingCoroutine());
    }

    IEnumerator ProfilingCoroutine()
    {
        isProfiling = true;
        snapshots.Clear();

        // 步骤1: 预热
        Debug.Log($"预热阶段 ({config.warmupTime}秒)...");
        yield return WarmupPhase();

        // 步骤2: 开始分析
        Debug.Log($"开始性能捕获 ({config.captureTime}秒)...");
        yield return ProfilingPhase();

        // 步骤3: 生成报告
        Debug.Log("生成性能报告...");
        GenerateReport();

        isProfiling = false;
        Debug.Log("性能分析完成");
    }

    IEnumerator WarmupPhase()
    {
        float endTime = Time.realtimeSinceStartup + config.warmupTime;

        while (Time.realtimeSinceStartup < endTime)
        {
            // 预热期间正常游戏逻辑
            yield return null;
        }
    }

    IEnumerator ProfilingPhase()
    {
        // 启用Profiler
        Profiler.enabled = true;
        if (config.captureDeepProfile)
        {
            Profiler.enableBinaryLog = true;
        }

        startTime = Time.realtimeSinceStartup;
        float endTime = startTime + config.captureTime;

        // 如果配置了内存快照，在开始前捕获一次
        if (config.captureMemorySnapshot)
        {
            CaptureMemorySnapshot("start");
        }

        // 主捕获循环
        while (Time.realtimeSinceStartup < endTime)
        {
            CaptureSnapshot();
            yield return null;
        }

        // 如果配置了内存快照，在结束后捕获一次
        if (config.captureMemorySnapshot)
        {
            CaptureMemorySnapshot("end");
        }

        // 禁用Profiler
        Profiler.enabled = false;
        if (config.captureDeepProfile)
        {
            Profiler.enableBinaryLog = false;
        }
    }

    void CaptureSnapshot()
    {
        var snapshot = new PerformanceSnapshot
        {
            timestamp = Time.realtimeSinceStartup - startTime,
            fps = 1.0f / Time.unscaledDeltaTime,
            frameTimeMs = Time.unscaledDeltaTime * 1000.0f,
            memoryMB = Profiler.GetTotalAllocatedMemoryLong() / (1024.0f * 1024.0f),
            drawCalls = UnityEngine.Rendering.RenderPipelineManager.currentPipeline?.GetDrawCalls() ?? 0,
            triangles = UnityEngine.Rendering.RenderPipelineManager.currentPipeline?.GetTriangles() ?? 0,
            vertices = UnityEngine.Rendering.RenderPipelineManager.currentPipeline?.GetVertices() ?? 0,
            cpuMainThreadMs = Profiler.GetRuntimeMemorySizeLong(UnityEngine.Profiling.ProfilerCategory.Internal) / 1024.0f,
            cpuRenderThreadMs = 0, // 需要特定API获取
            gpuTimeMs = 0 // 需要特定API获取
        };

        snapshots.Add(snapshot);
    }

    void CaptureMemorySnapshot(string label)
    {
        string snapshotPath = Path.Combine(config.outputPath, $"{reportFileName}_memory_{label}.snapshot");
        Profiler.WriteProfileDataSnapshot(snapshotPath);
        Debug.Log($"内存快照已保存: {snapshotPath}");
    }

    void GenerateReport()
    {
        if (snapshots.Count == 0)
        {
            Debug.LogWarning("无性能数据可生成报告");
            return;
        }

        // 计算统计数据
        PerformanceStats stats = CalculateStats();

        // 生成报告
        string report = GenerateReportText(stats);
        string csvData = GenerateCSVData();

        // 保存报告
        SaveReport(report, csvData, stats);

        // 检查性能问题
        CheckPerformanceIssues(stats);
    }

    PerformanceStats CalculateStats()
    {
        var stats = new PerformanceStats();

        // 基本统计
        stats.snapshotCount = snapshots.Count;
        stats.totalTime = snapshots[snapshots.Count - 1].timestamp;

        // 计算平均值、最小值、最大值
        float totalFPS = 0;
        float totalFrameTime = 0;
        float totalMemory = 0;
        int totalDrawCalls = 0;

        stats.minFPS = float.MaxValue;
        stats.maxFPS = float.MinValue;
        stats.minFrameTime = float.MaxValue;
        stats.maxFrameTime = float.MinValue;
        stats.minMemory = float.MaxValue;
        stats.maxMemory = float.MinValue;

        foreach (var snapshot in snapshots)
        {
            totalFPS += snapshot.fps;
            totalFrameTime += snapshot.frameTimeMs;
            totalMemory += snapshot.memoryMB;
            totalDrawCalls += snapshot.drawCalls;

            if (snapshot.fps < stats.minFPS) stats.minFPS = snapshot.fps;
            if (snapshot.fps > stats.maxFPS) stats.maxFPS = snapshot.fps;
            if (snapshot.frameTimeMs < stats.minFrameTime) stats.minFrameTime = snapshot.frameTimeMs;
            if (snapshot.frameTimeMs > stats.maxFrameTime) stats.maxFrameTime = snapshot.frameTimeMs;
            if (snapshot.memoryMB < stats.minMemory) stats.minMemory = snapshot.memoryMB;
            if (snapshot.memoryMB > stats.maxMemory) stats.maxMemory = snapshot.memoryMB;
        }

        stats.avgFPS = totalFPS / snapshots.Count;
        stats.avgFrameTime = totalFrameTime / snapshots.Count;
        stats.avgMemory = totalMemory / snapshots.Count;
        stats.avgDrawCalls = totalDrawCalls / snapshots.Count;

        // 计算百分位
        stats.fpsPercentiles = CalculatePercentiles(snapshots, s => s.fps);
        stats.frameTimePercentiles = CalculatePercentiles(snapshots, s => s.frameTimeMs);
        stats.memoryPercentiles = CalculatePercentiles(snapshots, s => s.memoryMB);

        return stats;
    }

    Dictionary<int, float> CalculatePercentiles(List<PerformanceSnapshot> data, System.Func<PerformanceSnapshot, float> selector)
    {
        var values = new List<float>();
        foreach (var snapshot in data)
        {
            values.Add(selector(snapshot));
        }
        values.Sort();

        var percentiles = new Dictionary<int, float>
        {
            { 50, GetPercentile(values, 50) },  // 中位数
            { 90, GetPercentile(values, 90) },  // 90%分位
            { 95, GetPercentile(values, 95) },  // 95%分位
            { 99, GetPercentile(values, 99) }   // 99%分位（最差1%）
        };

        return percentiles;
    }

    float GetPercentile(List<float> sortedValues, int percentile)
    {
        int index = (int)(sortedValues.Count * percentile / 100.0f);
        index = Mathf.Clamp(index, 0, sortedValues.Count - 1);
        return sortedValues[index];
    }

    string GenerateReportText(PerformanceStats stats)
    {
        var sb = new StringBuilder();

        sb.AppendLine($"性能测试报告: {config.testName}");
        sb.AppendLine($"生成时间: {System.DateTime.Now:yyyy-MM-dd HH:mm:ss}");
        sb.AppendLine($"测试时长: {stats.totalTime:F1}秒");
        sb.AppendLine($"快照数量: {stats.snapshotCount}");
        sb.AppendLine();

        sb.AppendLine("帧率分析 (FPS):");
        sb.AppendLine($"  平均: {stats.avgFPS:F1}");
        sb.AppendLine($"  最低: {stats.minFPS:F1}");
        sb.AppendLine($"  最高: {stats.maxFPS:F1}");
        sb.AppendLine($"  中位数 (P50): {stats.fpsPercentiles[50]:F1}");
        sb.AppendLine($"  90%分位 (P90): {stats.fpsPercentiles[90]:F1}");
        sb.AppendLine($"  99%分位 (P99): {stats.fpsPercentiles[99]:F1} (最差1%)");
        sb.AppendLine();

        sb.AppendLine("帧时间分析 (ms):");
        sb.AppendLine($"  平均: {stats.avgFrameTime:F2}ms");
        sb.AppendLine($"  最低: {stats.minFrameTime:F2}ms");
        sb.AppendLine($"  最高: {stats.maxFrameTime:F2}ms");
        sb.AppendLine($"  中位数 (P50): {stats.frameTimePercentiles[50]:F2}ms");
        sb.AppendLine($"  99%分位 (P99): {stats.frameTimePercentiles[99]:F2}ms (最差1%)");
        sb.AppendLine();

        sb.AppendLine("内存使用分析 (MB):");
        sb.AppendLine($"  平均: {stats.avgMemory:F1}MB");
        sb.AppendLine($"  最低: {stats.minMemory:F1}MB");
        sb.AppendLine($"  最高: {stats.maxMemory:F1}MB");
        sb.AppendLine($"  中位数 (P50): {stats.memoryPercentiles[50]:F1}MB");
        sb.AppendLine();

        sb.AppendLine("渲染统计:");
        sb.AppendLine($"  平均绘制调用: {stats.avgDrawCalls:F0}");
        sb.AppendLine();

        sb.AppendLine("性能评估:");
        sb.AppendLine($"  目标帧率: {config.targetFPS}FPS");
        sb.AppendLine($"  实际平均帧率: {stats.avgFPS:F1}FPS ({stats.avgFPS / config.targetFPS * 100:F1}% 目标)");
        sb.AppendLine($"  帧率稳定性: {stats.fpsPercentiles[99] / stats.avgFPS * 100:F1}% (P99/平均)");

        return sb.ToString();
    }

    string GenerateCSVData()
    {
        var sb = new StringBuilder();

        // CSV头部
        sb.AppendLine("Timestamp,FPS,FrameTimeMs,MemoryMB,DrawCalls,Triangles,Vertices");

        // 数据行
        foreach (var snapshot in snapshots)
        {
            sb.AppendLine($"{snapshot.timestamp:F3},{snapshot.fps:F1},{snapshot.frameTimeMs:F3},{snapshot.memoryMB:F1},{snapshot.drawCalls},{snapshot.triangles},{snapshot.vertices}");
        }

        return sb.ToString();
    }

    void SaveReport(string reportText, string csvData, PerformanceStats stats)
    {
        // 保存文本报告
        string reportPath = Path.Combine(config.outputPath, $"{reportFileName}_report.txt");

        // 保存CSV数据

        string json = JsonUtility.ToJson(stats, true);
        File.WriteAllText(jsonPath, json, Encoding.UTF8);

        Debug.Log($"性能报告已保存:");
        Debug.Log($"  文本报告: {reportPath}");
        Debug.Log($"  CSV数据: {csvPath}");
        Debug.Log($"  统计摘要: {jsonPath}");
    }

    void CheckPerformanceIssues(PerformanceStats stats)
    {
        List<string> issues = new List<string>();

        // 检查帧率问题
        if (stats.avgFPS < config.targetFPS * 0.8f) // 低于目标80%
        {
            issues.Add($"平均帧率({stats.avgFPS:F1}FPS)低于目标({config.targetFPS}FPS)的80%");
        }

        if (stats.minFPS < fpsCriticalThreshold)
        {
            issues.Add($"最低帧率({stats.minFPS:F1}FPS)低于严重阈值({fpsCriticalThreshold}FPS)");
        }
        else if (stats.minFPS < fpsWarningThreshold)
        {
            issues.Add($"最低帧率({stats.minFPS:F1}FPS)低于警告阈值({fpsWarningThreshold}FPS)");
        }

        // 检查内存问题
        if (stats.maxMemory > memoryWarningThresholdMB)
        {
            issues.Add($"峰值内存使用({stats.maxMemory:F1}MB)超过警告阈值({memoryWarningThresholdMB}MB)");
        }

        // 检查绘制调用
        if (stats.avgDrawCalls > drawCallWarningThreshold)
        {
            issues.Add($"平均绘制调用({stats.avgDrawCalls:F0})超过警告阈值({drawCallWarningThreshold})");
        }

        // 输出问题
        if (issues.Count > 0)
        {
            Debug.LogWarning("发现性能问题:");
            foreach (var issue in issues)
            {
                Debug.LogWarning($"  • {issue}");
            }
        }
        else
        {
            Debug.Log("✅ 未发现严重性能问题");
        }
    }
```

```python
# RenderDoc自动化分析脚本
import subprocess
import time
import os
import json
from pathlib import Path

class RenderDocAutomation:
    """RenderDoc自动化工具"""

    def __init__(self, renderdoc_path: str, game_executable: str):
        """
        初始化RenderDoc自动化

        Args:
            renderdoc_path: RenderDoc可执行文件路径
            game_executable: 要分析的游戏可执行文件路径
        """
        self.renderdoc_path = Path(renderdoc_path)
        self.game_executable = Path(game_executable)

        if not self.renderdoc_path.exists():
            raise FileNotFoundError(f"RenderDoc未找到: {renderdoc_path}")

        if not self.game_executable.exists():
            raise FileNotFoundError(f"游戏可执行文件未找到: {game_executable}")

    def capture_frame(self, output_dir: str, capture_point: str = "startup"):
        """
        捕获一帧

        Args:
            output_dir: 输出目录
            capture_point: 捕获点描述

        Returns:
            捕获文件路径
        """
        output_dir = Path(output_dir)
        output_dir.mkdir(parents=True, exist_ok=True)

        timestamp = time.strftime("%Y%m%d_%H%M%S")
        capture_file = output_dir / f"capture_{capture_point}_{timestamp}.rdc"

        # 构建RenderDoc命令行
        cmd = [
            str(self.renderdoc_path),
            "capture",  # 捕获命令
            "--opt", "APIValidation", "0",  # 禁用API验证提高性能
            "--opt", "AllowFullscreen", "1",  # 允许全屏
            "--opt", "AllowVSync", "0",  # 禁用垂直同步
            "--opt", "HookIntoChildren", "0",  # 不挂钩子进程
            "--opt", "CaptureAllCmdLists", "0",  # 不捕获所有命令列表
            "--opt", "RefAllResources", "0",  # 不引用所有资源
            "--opt", "CaptureCallstacks", "0",  # 不捕获调用栈
            "--opt", "CaptureCallstacksOnlyDraws", "0",  # 不限制调用栈捕获
            "--opt", "DelayForDebugger", "0",  # 不等待调试器
            "--opt", "VerifyMapWrites", "0",  # 不验证映射写入
            "--opt", "AutoStart", "1",  # 自动启动
            "--capture-file", str(capture_file),
            str(self.game_executable)
        ]

        print(f"开始捕获: {capture_point}")
        print(f"命令: {' '.join(cmd)}")

        try:
            # 启动游戏并捕获
            process = subprocess.Popen(
                cmd,
                stdout=subprocess.PIPE,
                stderr=subprocess.PIPE,
                text=True
            )

            # 等待游戏启动并运行一段时间
            time.sleep(10)  # 等待10秒让游戏启动

            # 在这里可以发送捕获触发（如果需要特定时刻）
            # 对于简单情况，RenderDoc会在启动时自动捕获

            # 等待游戏退出或超时
            timeout = 60  # 60秒超时
            start_time = time.time()

            while process.poll() is None:
                if time.time() - start_time > timeout:
                    print("超时，终止进程")
                    process.terminate()
                    break
                time.sleep(1)

            stdout, stderr = process.communicate()

            if process.returncode != 0:
                print(f"捕获过程异常，返回码: {process.returncode}")
                if stderr:
                    print(f"错误输出: {stderr}")

            if capture_file.exists():
                print(f"捕获成功: {capture_file}")
                return capture_file
            else:
                print("捕获失败: 未生成捕获文件")
                return None

        except Exception as e:
            print(f"捕获过程异常: {e}")
            return None

    def analyze_capture(self, capture_file: Path, output_dir: Path):
        """
        分析捕获文件

        Args:
            capture_file: 捕获文件路径
            output_dir: 输出目录
        """
        if not capture_file.exists():
            print(f"捕获文件不存在: {capture_file}")
            return

        output_dir.mkdir(parents=True, exist_ok=True)

        # 分析输出文件
        analysis_file = output_dir / f"{capture_file.stem}_analysis.json"
        summary_file = output_dir / f"{capture_file.stem}_summary.txt"

        # 使用RenderDoc命令行分析（RenderDoc的Python API更强大，但需要安装）
        # 这里使用简单的外部调用示例

        print(f"分析捕获文件: {capture_file}")

        # 实际实现可能需要使用RenderDoc的Python API
        # 以下为示例结构

        analysis_data = {
            "capture_file": str(capture_file),
            "analysis_time": time.strftime("%Y-%m-%d %H:%M:%S"),
            "draw_calls": self.analyze_draw_calls(capture_file),
            "textures": self.analyze_textures(capture_file),
            "shaders": self.analyze_shaders(capture_file),
            "performance": self.analyze_performance(capture_file)
        }

        # 保存分析结果
        with open(analysis_file, 'w', encoding='utf-8') as f:
            json.dump(analysis_data, f, indent=2, ensure_ascii=False)

        # 生成摘要报告
        summary = self.generate_summary(analysis_data)
        with open(summary_file, 'w', encoding='utf-8') as f:
            f.write(summary)

        print(f"分析完成:")
        print(f"  分析数据: {analysis_file}")
        print(f"  摘要报告: {summary_file}")

        return analysis_data

    def generate_summary(self, analysis_data: dict) -> str:
        """生成摘要报告"""
        draw = analysis_data["draw_calls"]
        tex = analysis_data["textures"]
        shader = analysis_data["shaders"]
        perf = analysis_data["performance"]

        summary = f"""RenderDoc分析摘要
{'='*50}

基本统计:
  绘制调用总数: {draw['total']}
  纹理总数: {tex['total_textures']} ({tex['total_memory_mb']:.1f} MB)
  着色器总数: {shader['total_shaders']}

  GPU时间: {perf['gpu_time_ms']:.1f} ms/帧
  CPU时间: {perf['cpu_time_ms']:.1f} ms/帧

关键发现:
  最大纹理: {tex['largest_texture']['name']} ({tex['largest_texture']['size']}, {tex['largest_texture']['memory_mb']:.1f} MB)

        for i, rec in enumerate(perf['recommendations'], 1):
            summary += f"  {i}. {rec}\n"

        summary += f"\n{'='*50}\n分析时间: {analysis_data['analysis_time']}\n"

        return summary

    def batch_analysis(self, test_scenarios: list, output_base_dir: str):
        """
        批量分析多个测试场景

        Args:
            test_scenarios: 测试场景列表
            output_base_dir: 输出基础目录
        """
        output_base = Path(output_base_dir)

        all_results = {}

        for scenario in test_scenarios:
            print(f"\n处理场景: {scenario['name']}")

            # 场景特定输出目录
            scene_dir = output_base / scenario['name']
            scene_dir.mkdir(parents=True, exist_ok=True)

            # 捕获帧
            capture_file = self.capture_frame(
                str(scene_dir),
                scenario.get('capture_point', 'default')
            )

            if capture_file:
                # 分析捕获
                analysis = self.analyze_capture(capture_file, scene_dir)

                if analysis:
                    all_results[scenario['name']] = {
                        'scenario': scenario,
                        'analysis': analysis,
                        'capture_file': str(capture_file)
                    }

            # 场景间延迟（让系统冷却）
            time.sleep(5)

        # 生成比较报告
        self.generate_comparison_report(all_results, output_base)

        return all_results

    def generate_comparison_report(self, all_results: dict, output_dir: Path):
        """生成场景比较报告"""
        comparison_file = output_dir / "scenario_comparison.md"

            f.write("## 性能指标汇总\n\n")
            f.write("| 场景 | 绘制调用 | 纹理内存(MB) | GPU时间(ms) | 瓶颈 |\n")
            f.write("|------|----------|---------------|-------------|------|\n")

            for scene_name, result in all_results.items():
                analysis = result['analysis']
                draw = analysis['draw_calls']
                tex = analysis['textures']
                perf = analysis['performance']

                f.write(f"| {scene_name} | {draw['total']} | {tex['total_memory_mb']:.1f} | {perf['gpu_time_ms']:.1f} | {perf['bottleneck']} |\n")

            f.write("\n## 详细分析\n\n")

            for scene_name, result in all_results.items():
                f.write(f"### {scene_name}\n\n")
                f.write(f"**描述**: {result['scenario'].get('description', '无描述')}\n\n")

                analysis = result['analysis']
                perf = analysis['performance']

                f.write("**关键指标**:\n")
                f.write(f"- GPU时间: {perf['gpu_time_ms']:.1f} ms\n")
                f.write(f"- CPU时间: {perf['cpu_time_ms']:.1f} ms\n")
                f.write(f"- 瓶颈: {perf['bottleneck']}\n\n")

                f.write("**优化建议**:\n")
                for rec in perf['recommendations']:
                    f.write(f"- {rec}\n")
                f.write("\n")

        print(f"比较报告已生成: {comparison_file}")
```

```python
        cursor.execute("""
            CREATE TABLE IF NOT EXISTS performance_metrics (
                id INTEGER PRIMARY KEY AUTOINCREMENT,
                run_id TEXT NOT NULL,
                timestamp TEXT NOT NULL,
                metric_name TEXT NOT NULL,
                metric_value REAL NOT NULL,
                unit TEXT NOT NULL,
                scenario TEXT,
                FOREIGN KEY (run_id) REFERENCES test_runs (run_id)
            )
        """)

        # 代码覆盖率表
        cursor.execute("""
            CREATE TABLE IF NOT EXISTS code_coverage (
                id INTEGER PRIMARY KEY AUTOINCREMENT,
                run_id TEXT NOT NULL,
                timestamp TEXT NOT NULL,
                assembly_name TEXT NOT NULL,
                line_coverage REAL NOT NULL,
                branch_coverage REAL NOT NULL,
                covered_lines INTEGER NOT NULL,
                total_lines INTEGER NOT NULL,
                FOREIGN KEY (run_id) REFERENCES test_runs (run_id)
            )
        """)

        self.conn.commit()

    def add_test_run(self, run_data: dict):
        """添加测试运行记录"""
        cursor = self.conn.cursor()

        cursor.execute("""
            INSERT INTO test_runs (
                run_id, timestamp, branch, commit_hash, test_type,
                total_tests, passed, failed, skipped, duration_seconds, success_rate
            ) VALUES (?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?)
        """, (
            run_data['run_id'],
            run_data['timestamp'],
            run_data['branch'],
            run_data['commit_hash'],
            run_data['test_type'],
            run_data['total_tests'],
            run_data['passed'],
            run_data['failed'],
            run_data['skipped'],
            run_data['duration_seconds'],
            run_data['success_rate']
        ))

        # 添加测试用例
        for test_case in run_data.get('test_cases', []):
            cursor.execute("""
                INSERT INTO test_cases (
                    run_id, test_name, test_class, status,
                    duration_seconds, error_message, stack_trace
                ) VALUES (?, ?, ?, ?, ?, ?, ?)
            """, (
                run_data['run_id'],
                test_case['test_name'],
                test_case.get('test_class'),
                test_case['status'],
                test_case.get('duration_seconds'),
                test_case.get('error_message'),
                test_case.get('stack_trace')
            ))

        self.conn.commit()

    def get_test_summary(self, days: int = 7) -> dict:
        """获取测试摘要"""
        cursor = self.conn.cursor()

        # 总体统计
        cursor.execute("""
            SELECT
                COUNT(*) as total_runs,
                AVG(success_rate) as avg_success_rate,
                SUM(total_tests) as total_tests_executed,
                SUM(passed) as total_passed,
                SUM(failed) as total_failed
            FROM test_runs
            WHERE timestamp >= datetime('now', ?)
        """, (f'-{days} days',))

        overall = cursor.fetchone()

        # 按测试类型统计
        cursor.execute("""
            SELECT
                test_type,
                COUNT(*) as run_count,
                AVG(success_rate) as avg_success_rate,
                AVG(duration_seconds) as avg_duration
            FROM test_runs
            WHERE timestamp >= datetime('now', ?)
            GROUP BY test_type
        """, (f'-{days} days',))

        by_type = cursor.fetchall()

        # 趋势数据
        cursor.execute("""
            SELECT
                date(timestamp) as date,
                test_type,
                AVG(success_rate) as avg_success_rate
            FROM test_runs
            WHERE timestamp >= datetime('now', ?)
            GROUP BY date(timestamp), test_type
            ORDER BY date(timestamp)
        """, (f'-{days} days',))

        trend = cursor.fetchall()

        return {
            'overall': {
                'total_runs': overall[0],
                'avg_success_rate': overall[1],
                'total_tests_executed': overall[2],
                'total_passed': overall[3],
                'total_failed': overall[4]
            },
            'by_type': [
                {
                    'test_type': row[0],
                    'run_count': row[1],
                    'avg_success_rate': row[2],
                    'avg_duration': row[3]
                }
                for row in by_type
            ],
            'trend': [
                {
                    'date': row[0],
                    'test_type': row[1],
                    'avg_success_rate': row[2]
                }
                for row in trend
            ]
        }

    def get_failing_tests(self, limit: int = 10) -> list:
        """获取最近失败最多的测试"""
        cursor = self.conn.cursor()

        cursor.execute("""
            SELECT
                test_name,
                test_class,
                COUNT(*) as failure_count,
                GROUP_CONCAT(DISTINCT run_id) as failed_runs
            FROM test_cases
            WHERE status = 'Failed'
            GROUP BY test_name, test_class
            ORDER BY failure_count DESC
            LIMIT ?
        """, (limit,))

        return [
            {
                'test_name': row[0],
                'test_class': row[1],
                'failure_count': row[2],
                'failed_runs': row[3].split(',')
            }
            for row in cursor.fetchall()
        ]

    def create_dashboard(self):
        """创建Streamlit仪表板"""
        st.set_page_config(
            page_title="游戏测试监控仪表板",
            page_icon="🎮",
            layout="wide"
        )

        st.title("🎮 游戏测试监控仪表板")

        # 侧边栏：筛选选项
        with st.sidebar:
            st.header("筛选选项")

            days = st.slider(
                "查看天数",
                min_value=1,
                max_value=30,
                value=7
            )

            test_type = st.multiselect(
                "测试类型",
                options=["Unit", "Integration", "Functional", "Performance"],
                default=["Unit", "Integration", "Functional", "Performance"]
            )

            branch = st.multiselect(
                "分支",
                options=self.get_branches(),
                default=["main", "develop"]
            )

        # 获取数据
        summary = self.get_test_summary(days)

        # 概览指标
        col1, col2, col3, col4 = st.columns(4)

        with col1:
            st.metric(
                label="总测试运行",
                value=summary['overall']['total_runs'],
                delta=None
            )

        with col2:
            success_rate = summary['overall']['avg_success_rate']
            st.metric(
                label="平均成功率",
                value=f"{success_rate:.1%}",
                delta=None
            )

        with col3:
            st.metric(
                label="总测试用例",
                value=summary['overall']['total_tests_executed'],
                delta=None
            )

        with col4:
            failure_rate = (summary['overall']['total_failed'] /
                          summary['overall']['total_tests_executed'])
            st.metric(
                label="失败率",
                value=f"{failure_rate:.1%}",
                delta=None
            )

        # 图表区域
        tab1, tab2, tab3, tab4 = st.tabs([
            "趋势分析",
            "测试类型分布",
            "失败测试",
            "性能指标"
        ])

        with tab1:
            # 成功率趋势图
            df_trend = pd.DataFrame(summary['trend'])

            if not df_trend.empty:
                fig = px.line(
                    df_trend,
                    x='date',
                    y='avg_success_rate',
                    color='test_type',
                    title="测试成功率趋势",
                    labels={
                        'date': '日期',
                        'avg_success_rate': '平均成功率',
                        'test_type': '测试类型'
                    }
                )

                fig.update_layout(
                    yaxis_tickformat=".0%",
                    hovermode="x unified"
                )

                st.plotly_chart(fig, use_container_width=True)
            else:
                st.info("暂无趋势数据")

        with tab2:
            # 测试类型分布
            df_by_type = pd.DataFrame(summary['by_type'])

            if not df_by_type.empty:
                col1, col2 = st.columns(2)

                with col1:
                    # 运行次数饼图
                    fig1 = px.pie(
                        df_by_type,
                        values='run_count',
                        names='test_type',
                        title="测试运行次数分布",
                        hole=0.3
                    )
                    st.plotly_chart(fig1, use_container_width=True)

                with col2:
                    # 成功率柱状图
                    fig2 = px.bar(
                        df_by_type,
                        x='test_type',
                        y='avg_success_rate',
                        title="各测试类型平均成功率",
                        color='test_type',
                        text_auto='.1%'
                    )

                    fig2.update_layout(
                        yaxis_tickformat=".0%",
                        showlegend=False
                    )

                    st.plotly_chart(fig2, use_container_width=True)
            else:
                st.info("暂无测试类型数据")

        with tab3:
            # 失败测试列表
            failing_tests = self.get_failing_tests(10)

            if failing_tests:
                df_failing = pd.DataFrame(failing_tests)

                st.dataframe(
                    df_failing,
                    column_config={
                        'test_name': '测试名称',
                        'test_class': '测试类',
                        'failure_count': '失败次数',
                        'failed_runs': st.column_config.ListColumn(
                            '失败运行ID',
                            help="该测试失败的运行ID列表"
                        )
                    },
                    hide_index=True,
                    use_container_width=True
                )
            else:
                st.success("🎉 最近没有频繁失败的测试！")

        with tab4:
            # 性能指标（如果有数据）
            cursor = self.conn.cursor()
            cursor.execute("""
                SELECT
                    date(timestamp) as date,
                    metric_name,
                    AVG(metric_value) as avg_value
                FROM performance_metrics
                WHERE timestamp >= datetime('now', ?)
                GROUP BY date(timestamp), metric_name
                ORDER BY date(timestamp)
            """, (f'-{days} days',))

            perf_data = cursor.fetchall()

            if perf_data:
                df_perf = pd.DataFrame(perf_data,
                                     columns=['date', 'metric_name', 'avg_value'])

                # 性能指标选择
                metrics = df_perf['metric_name'].unique()
                selected_metrics = st.multiselect(
                    "选择性能指标",
                    options=metrics,
                    default=metrics[:3] if len(metrics) > 3 else metrics
                )

                if selected_metrics:
                    df_filtered = df_perf[df_perf['metric_name'].isin(selected_metrics)]

                    fig = px.line(
                        df_filtered,
                        x='date',
                        y='avg_value',
                        color='metric_name',
                        title="性能指标趋势",
                        labels={
                            'date': '日期',
                            'avg_value': '指标值',
                            'metric_name': '指标名称'
                        }
                    )

                    st.plotly_chart(fig, use_container_width=True)
            else:
                st.info("暂无性能指标数据")

        # 详细数据表
        st.header("详细测试运行记录")

        cursor.execute("""
            SELECT
                run_id,
                timestamp,
                branch,
                test_type,
                total_tests,
                passed,
                failed,
                skipped,
                duration_seconds,
                success_rate
            FROM test_runs
            WHERE timestamp >= datetime('now', ?)
            ORDER BY timestamp DESC
        """, (f'-{days} days',))

        runs = cursor.fetchall()

        if runs:
            df_runs = pd.DataFrame(runs, columns=[
                '运行ID', '时间', '分支', '测试类型', '总测试数',
                '通过', '失败', '跳过', '持续时间(秒)', '成功率'
            ])

            st.dataframe(
                df_runs,
                column_config={
                    '成功率': st.column_config.ProgressColumn(
                        '成功率',
                        help="测试成功率",
                        format="%.1f%%",
                        min_value=0,
                        max_value=100
                    )
                },
                hide_index=True,
                use_container_width=True
            )
        else:
            st.info("暂无测试运行记录")

    def get_branches(self) -> list:
        """获取所有分支"""
        cursor = self.conn.cursor()
        cursor.execute("SELECT DISTINCT branch FROM test_runs")
        return [row[0] for row in cursor.fetchall()]
```
