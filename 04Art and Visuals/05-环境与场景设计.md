# 环境与场景设计

## 概述

游戏环境是玩家沉浸其中的虚拟世界，它不仅仅是背景装饰，更是叙事、游戏机制和情感体验的重要载体。优秀的环境设计能够引导玩家、讲述故事、营造氛围，甚至成为游戏玩法的一部分。本章将深入探讨游戏环境与场景设计的艺术与技术，特别关注独立游戏开发中的资源限制和实用解决方案。

**学习目标**：
- 理解环境设计在游戏体验中的多重作用
- 掌握环境艺术设计的核心原则和技术
- 学习模块化设计和可重用资产的高效工作流
- 掌握氛围营造和视觉叙事的技巧
- 了解环境性能优化和资源管理策略
- 能够为游戏项目制定完整的环境美术生产计划

## 1、环境设计基础

### 1.1 环境设计的核心功能

**环境的五个关键作用**：

1. **游戏空间定义**：为玩法提供物理边界和交互空间
2. **玩家引导**：通过视觉线索引导玩家探索和前进
3. **故事讲述**：通过环境细节传达世界观和叙事
4. **氛围营造**：创造情感体验和沉浸感
5. **视觉框架**：为游戏动作提供视觉背景和舞台

**独立游戏环境设计理念**：
> **核心原则**：在独立游戏中，环境设计不应追求庞大的规模或极致的细节，而应专注于**一致性、氛围和功能性**。一个风格统一、精心设计的小环境，远比一个庞大但空洞的世界更有感染力。

### 1.3 环境设计原则

**空间设计的心理学**：

**环境空间感知原则**：
1. **比例与尺度**：不同大小的空间创造不同的心理感受
2. **封闭与开放**：封闭空间提供安全感，开放空间激发探索欲
3. **复杂与简单**：复杂环境激发好奇，简单环境提供清晰导向
4. **自然与人造**：自然元素舒缓情绪，人造结构传达秩序

**Unity模块化工作流示例**：
```csharp
// Unity模块化环境系统
using UnityEngine;
using System.Collections.Generic;

public class ModularEnvironmentBuilder : MonoBehaviour
{
    public GameObject[] wallModules;
    public GameObject[] floorModules;
    public GameObject[] connectorModules;
    public GameObject[] detailModules;

    public int gridSizeX = 10;
    public int gridSizeZ = 10;
    public float gridSpacing = 2.0f;

    private GameObject[,] environmentGrid;

    void Start()
    {
        BuildEnvironment();
    }

    void BuildEnvironment()
    {
        environmentGrid = new GameObject[gridSizeX, gridSizeZ];

        // 构建地板
        BuildFloorGrid();

        // 构建墙壁
        BuildWalls();

        // 添加细节
        AddDetails();

        // 优化环境
        OptimizeEnvironment();
    }

    void BuildFloorGrid()
    {
        for (int x = 0; x < gridSizeX; x++)
        {
            for (int z = 0; z < gridSizeZ; z++)
            {
                // 选择地板模块（考虑边缘情况）
                GameObject floorPrefab = SelectFloorModule(x, z);

                Vector3 position = new Vector3(
                    x * gridSpacing,
                    0,
                    z * gridSpacing
                );

                GameObject floorInstance = Instantiate(
                    floorPrefab,
                    position,
                    Quaternion.identity,
                    transform
                );

                environmentGrid[x, z] = floorInstance;
            }
        }
    }

    GameObject SelectFloorModule(int x, int z)
    {
        // 基本选择逻辑
        bool isEdge = (x == 0 || x == gridSizeX - 1 || z == 0 || z == gridSizeZ - 1);

        if (isEdge)
        {
            // 边缘使用特殊地板
            return floorModules[1]; // 边缘变体
        }
        else
        {
            // 内部随机选择，但控制重复
            return GetWeightedRandomFloor();
        }
    }

    GameObject GetWeightedRandomFloor()
    {
        // 基于已经使用的类型加权随机
        // 确保不会过度重复相同模块
        float[] weights = new float[floorModules.Length];

        for (int i = 0; i < weights.Length; i++)
        {
            weights[i] = 1.0f; // 基础权重

            // 减少已过度使用的模块权重
            int count = CountModuleType(floorModules[i]);
            if (count > 5) // 如果已使用超过5次
            {
                weights[i] = 0.5f;
            }
            if (count > 10) // 如果已使用超过10次
            {
                weights[i] = 0.2f;
            }
        }

        // 加权随机选择
        float totalWeight = 0;
        foreach (float w in weights) totalWeight += w;

        float randomValue = Random.Range(0, totalWeight);
        float currentSum = 0;

        for (int i = 0; i < floorModules.Length; i++)
        {
            currentSum += weights[i];
            if (randomValue <= currentSum)
            {
                return floorModules[i];
            }
        }

        return floorModules[0]; // 默认
    }

    int CountModuleType(GameObject moduleType)
    {
        // 计算场景中已使用的该模块数量
        int count = 0;
        foreach (GameObject obj in environmentGrid)
        {
            if (obj != null && obj.name.Contains(moduleType.name))
            {
                count++;
            }
        }
        return count;
    }

    void BuildWalls()
    {
        // 在外围构建墙壁
        for (int x = 0; x < gridSizeX; x++)
        {
            // 前墙和后墙
            CreateWall(x, 0, 0); // 前墙
            CreateWall(x, gridSizeZ - 1, 180); // 后墙
        }

        for (int z = 0; z < gridSizeZ; z++)
        {
            // 左墙和右墙
            CreateWall(0, z, 90); // 左墙
            CreateWall(gridSizeX - 1, z, -90); // 右墙
        }
    }

    void CreateWall(int gridX, int gridZ, float rotationY)
    {
        Vector3 position = new Vector3(
            gridX * gridSpacing,
            1.5f, // 墙壁高度
            gridZ * gridSpacing
        );

        Quaternion rotation = Quaternion.Euler(0, rotationY, 0);

        // 随机选择墙壁变体
        int wallIndex = Random.Range(0, wallModules.Length);
        GameObject wallInstance = Instantiate(
            wallModules[wallIndex],
            position,
            rotation,
            transform
        );

        // 检查是否需要特殊连接模块
        AddConnectorIfNeeded(gridX, gridZ, rotationY);
    }

    void AddConnectorIfNeeded(int gridX, int gridZ, float rotationY)
    {
        // 在角落添加连接器
        bool isCorner = (gridX == 0 && gridZ == 0) ||
                       (gridX == 0 && gridZ == gridSizeZ - 1) ||
                       (gridX == gridSizeX - 1 && gridZ == 0) ||
                       (gridX == gridSizeX - 1 && gridZ == gridSizeZ - 1);

        if (isCorner)
        {
            Vector3 cornerPosition = new Vector3(
                gridX * gridSpacing,
                1.5f,
                gridZ * gridSpacing
            );

            // 选择角落连接器
            GameObject cornerConnector = connectorModules[0];
            Instantiate(cornerConnector, cornerPosition, Quaternion.identity, transform);
        }
    }

    void AddDetails()
    {
        // 随机添加细节道具
        int detailCount = (gridSizeX * gridSizeZ) / 4; // 每4格一个细节

        for (int i = 0; i < detailCount; i++)
        {
```

```csharp
public class SimpleWeatherSystem : MonoBehaviour
{
    [System.Serializable]
    public class WeatherPreset
    {
        public string weatherName;
        public Color skyColor;
        public Color fogColor;
        public float fogDensity;
        public GameObject particleEffect;
        public AudioClip ambientSound;
        public float soundVolume = 0.5f;
        public float transitionTime = 5.0f;
    }

    public WeatherPreset[] weatherPresets;
    public Light directionalLight;

    private WeatherPreset currentWeather;
    private WeatherPreset targetWeather;
    private float transitionProgress = 0f;
    private bool isTransitioning = false;

    private AudioSource audioSource;
    private GameObject currentParticles;

    void Start()
    {
        audioSource = gameObject.AddComponent<AudioSource>();
        audioSource.loop = true;
        audioSource.spatialBlend = 0f; // 2D声音

        // 初始天气
        if (weatherPresets.Length > 0)
        {
            SetWeather(weatherPresets[0]);
        }
    }

    void Update()
    {
        // 处理天气过渡
        if (isTransitioning && targetWeather != null)
        {
            transitionProgress += Time.deltaTime / targetWeather.transitionTime;

            // 插值过渡
            LerpWeather(currentWeather, targetWeather, transitionProgress);

            if (transitionProgress >= 1.0f)
            {
                // 过渡完成
                currentWeather = targetWeather;
                isTransitioning = false;
                transitionProgress = 0f;
            }
        }

        // 测试：按键切换天气
        if (Input.GetKeyDown(KeyCode.Alpha1) && weatherPresets.Length > 0)
            ChangeWeather(weatherPresets[0]);
        if (Input.GetKeyDown(KeyCode.Alpha2) && weatherPresets.Length > 1)
            ChangeWeather(weatherPresets[1]);
        if (Input.GetKeyDown(KeyCode.Alpha3) && weatherPresets.Length > 2)
            ChangeWeather(weatherPresets[2]);
    }

    public void ChangeWeather(WeatherPreset newWeather)
    {
        if (isTransitioning)
        {
            // 如果已经在过渡中，快速完成当前过渡
            transitionProgress = 1.0f;
        }

        targetWeather = newWeather;
        isTransitioning = true;
        transitionProgress = 0f;

        Debug.Log($"开始切换天气到: {newWeather.weatherName}");
    }

    void SetWeather(WeatherPreset weather)
    {
        // 立即设置天气，无过渡
        RenderSettings.fogColor = weather.fogColor;
        RenderSettings.fogDensity = weather.fogDensity;
        RenderSettings.ambientLight = weather.skyColor;

        if (directionalLight != null)
        {
            // 根据天气调整阳光
            directionalLight.intensity = weather.weatherName == "Clear" ? 1.0f : 0.5f;
            directionalLight.color = weather.skyColor;
        }

        // 粒子效果
        if (currentParticles != null)
        {
            Destroy(currentParticles);
        }

        if (weather.particleEffect != null)
        {
            currentParticles = Instantiate(weather.particleEffect, transform.position, Quaternion.identity);
            currentParticles.transform.parent = transform;
        }

        // 环境音
        if (weather.ambientSound != null)
        {
            audioSource.clip = weather.ambientSound;
            audioSource.volume = weather.soundVolume;
            audioSource.Play();
        }

        currentWeather = weather;

        Debug.Log($"天气已设置为: {weather.weatherName}");
    }

    void LerpWeather(WeatherPreset from, WeatherPreset to, float t)
    {
        if (from == null || to == null) return;

        // 插值颜色和密度
        RenderSettings.fogColor = Color.Lerp(from.fogColor, to.fogColor, t);
        RenderSettings.fogDensity = Mathf.Lerp(from.fogDensity, to.fogDensity, t);
        RenderSettings.ambientLight = Color.Lerp(from.skyColor, to.skyColor, t);

        if (directionalLight != null)
        {
            directionalLight.color = Color.Lerp(from.skyColor, to.skyColor, t);
        }

        // 交叉淡入淡出粒子效果（简化）
        if (t > 0.5f && currentParticles != null &&
            from.particleEffect != to.particleEffect)
        {
            // 在新的粒子效果准备好后移除旧的
            if (t > 0.5f && currentParticles.name.Contains(from.particleEffect.name))
            {
                Destroy(currentParticles);
                if (to.particleEffect != null)
                {
                    currentParticles = Instantiate(to.particleEffect, transform.position, Quaternion.identity);
                    currentParticles.transform.parent = transform;
                }
            }
        }

        // 交叉淡入淡出音频
        if (from.ambientSound != to.ambientSound)
        {
            if (t < 0.5f)
            {
                audioSource.volume = Mathf.Lerp(from.soundVolume, 0, t * 2);
            }
            else
            {
                if (audioSource.clip != to.ambientSound)
                {
                    audioSource.clip = to.ambientSound;
                    audioSource.Play();
                }
                audioSource.volume = Mathf.Lerp(0, to.soundVolume, (t - 0.5f) * 2);
            }
        }
        else
        {
            audioSource.volume = Mathf.Lerp(from.soundVolume, to.soundVolume, t);
        }
    }
```

```csharp
// Unity简单破坏系统
using UnityEngine;
using System.Collections.Generic;

public class DestructibleEnvironment : MonoBehaviour
{
    [System.Serializable]
    public class DestructionStage
    {
        public GameObject stagePrefab; // 该阶段的视觉表现
        public float healthThreshold;  // 切换到该阶段的生命值阈值
        public bool isFinalStage = false;
    }

    public float maxHealth = 100f;
    public float currentHealth = 100f;
    public DestructionStage[] destructionStages;
    public GameObject destructionEffect; // 破坏时的粒子效果
    public AudioClip destructionSound;

    private int currentStageIndex = 0;
    private GameObject currentStageObject;

    void Start()
    {
        InitializeDestructible();
    }

    void InitializeDestructible()
    {
        if (destructionStages.Length > 0)
        {
            // 创建初始阶段
            currentStageObject = Instantiate(
                destructionStages[0].stagePrefab,
                transform.position,
                transform.rotation,
                transform
            );
        }
    }

    public void TakeDamage(float damage)
    {
        if (currentHealth <= 0) return; // 已破坏

        currentHealth -= damage;

        // 检查是否需要切换到下一阶段
        CheckStageTransition();

        // 如果生命值归零，完全破坏
        if (currentHealth <= 0)
        {
            DestroyCompletely();
        }
    }

    void CheckStageTransition()
    {
        float healthPercentage = currentHealth / maxHealth;

        for (int i = destructionStages.Length - 1; i >= 0; i--)
        {
            if (healthPercentage <= destructionStages[i].healthThreshold && i > currentStageIndex)
            {
                TransitionToStage(i);
                break;
            }
        }
    }

    void TransitionToStage(int stageIndex)
    {
        if (stageIndex >= destructionStages.Length) return;

        // 销毁当前阶段对象
        if (currentStageObject != null)
        {
            Destroy(currentStageObject);
        }

        // 创建新阶段对象
        currentStageObject = Instantiate(
            destructionStages[stageIndex].stagePrefab,
            transform.position,
            transform.rotation,
            transform
        );

        currentStageIndex = stageIndex;

        // 播放阶段过渡效果
        PlayTransitionEffect();

        Debug.Log($"切换到破坏阶段 {stageIndex}");
    }

    void DestroyCompletely()
    {
        // 播放破坏效果
        if (destructionEffect != null)
        {
            Instantiate(destructionEffect, transform.position, Quaternion.identity);
        }

        if (destructionSound != null)
        {
            AudioSource.PlayClipAtPoint(destructionSound, transform.position);
        }

        // 禁用碰撞体
        Collider col = GetComponent<Collider>();
        if (col != null) col.enabled = false;

        // 隐藏对象
        if (currentStageObject != null)
        {
            currentStageObject.SetActive(false);
        }

        // 标记为已破坏
        enabled = false;

        Debug.Log($"{gameObject.name} 已完全破坏");
    }

    void PlayTransitionEffect()
    {
        // 简单的粒子效果
        GameObject effect = new GameObject("TransitionEffect");
        effect.transform.position = transform.position;

        ParticleSystem ps = effect.AddComponent<ParticleSystem>();

        var main = ps.main;
        main.startSize = 0.1f;
        main.startSpeed = 2.0f;
        main.startLifetime = 0.5f;
        main.maxParticles = 20;

        var emission = ps.emission;
        emission.rateOverTime = 50;

        var shape = ps.shape;
        shape.shapeType = ParticleSystemShapeType.Sphere;
        shape.radius = 0.5f;

        // 自动销毁效果
        Destroy(effect, 1.0f);
    }

    // 可视化当前状态
    void OnDrawGizmosSelected()
    {
        // 绘制生命值条
        float healthPercentage = currentHealth / maxHealth;

        Vector3 startPos = transform.position + Vector3.up * 1.5f;
        Vector3 endPos = startPos + Vector3.right * 2f;

        // 背景条
        Gizmos.color = Color.gray;
        Gizmos.DrawLine(startPos, endPos);

        // 生命值条
        Gizmos.color = Color.Lerp(Color.red, Color.green, healthPercentage);
        Vector3 healthEnd = Vector3.Lerp(startPos, endPos, healthPercentage);
        Gizmos.DrawLine(startPos, healthEnd);

        // 绘制当前阶段指示器
        if (destructionStages != null && currentStageIndex < destructionStages.Length)
        {
            Gizmos.color = Color.yellow;
            Gizmos.DrawWireSphere(transform.position, 0.3f + currentStageIndex * 0.1f);
        }
    }
}
```

**资产优先级管理**：

```csharp
// Unity资产加载优先级系统
using UnityEngine;
using System.Collections.Generic;

public class AssetPriorityLoader : MonoBehaviour
{
    [System.Serializable]
    public class AssetLoadRequest
    {
        public string assetPath;
        public System.Action<Object> onLoaded;
        public int priority; // 0-100，越高越优先
        public float loadDistance; // 加载距离
        public bool isEssential; // 是否必需（阻塞加载）
    }

    private List<AssetLoadRequest> loadQueue = new List<AssetLoadRequest>();
    private List<AssetLoadRequest> activeLoads = new List<AssetLoadRequest>();
    private Dictionary<string, Object> loadedAssets = new Dictionary<string, Object>();

    public int maxConcurrentLoads = 3;
    public float updateInterval = 0.1f;

    private Transform playerTransform;
    private float updateTimer = 0f;

    void Start()
    {
        playerTransform = Camera.main?.transform;

        if (playerTransform == null)
        {
            GameObject player = GameObject.FindGameObjectWithTag("Player");
            if (player != null) playerTransform = player.transform;
        }
    }

    void Update()
    {
        updateTimer += Time.deltaTime;
        if (updateTimer >= updateInterval)
        {
            UpdateLoadQueue();
            updateTimer = 0f;
        }
    }

    public void RequestAssetLoad(AssetLoadRequest request)
    {
        // 检查是否已加载
        if (loadedAssets.ContainsKey(request.assetPath))
        {
            request.onLoaded?.Invoke(loadedAssets[request.assetPath]);
            return;
        }

        // 添加到队列
        loadQueue.Add(request);

        // 按优先级排序
        loadQueue.Sort((a, b) => b.priority.CompareTo(a.priority));
    }

    void UpdateLoadQueue()
    {
        // 移除已完成或无效的请求
        loadQueue.RemoveAll(request =>
            request == null ||
            (request.loadDistance > 0 && playerTransform != null &&
             Vector3.Distance(playerTransform.position, transform.position) > request.loadDistance)
        );

        // 开始新的加载
        while (activeLoads.Count < maxConcurrentLoads && loadQueue.Count > 0)
        {
            AssetLoadRequest request = loadQueue[0];
            loadQueue.RemoveAt(0);

            StartLoad(request);
        }
    }

    void StartLoad(AssetLoadRequest request)
    {
        activeLoads.Add(request);

        if (request.isEssential)
        {
            // 同步加载
            Object asset = Resources.Load(request.assetPath);
            OnAssetLoaded(request, asset);
        }
        else
        {
            // 异步加载
            ResourceRequest asyncOperation = Resources.LoadAsync(request.assetPath);
            StartCoroutine(WaitForLoad(asyncOperation, request));
        }
    }

    System.Collections.IEnumerator WaitForLoad(ResourceRequest operation, AssetLoadRequest request)
    {
        yield return operation;

        OnAssetLoaded(request, operation.asset);
    }

    void OnAssetLoaded(AssetLoadRequest request, Object asset)
    {
        // 从活动加载列表中移除
        activeLoads.Remove(request);

        if (asset != null)
        {
            // 缓存资产
            if (!loadedAssets.ContainsKey(request.assetPath))
            {
                loadedAssets[request.assetPath] = asset;
            }

            // 调用回调
            request.onLoaded?.Invoke(asset);
        }
        else
        {
            Debug.LogWarning($"资产加载失败: {request.assetPath}");
        }
    }

    public void UnloadAsset(string assetPath, bool force = false)
    {
        if (loadedAssets.ContainsKey(assetPath))
        {
            // 检查是否有活跃请求使用此资产
            bool inUse = activeLoads.Exists(request => request.assetPath == assetPath);

            if (!inUse || force)
            {
                Resources.UnloadAsset(loadedAssets[assetPath]);
                loadedAssets.Remove(assetPath);
            }
        }
    }

    public void UnloadUnusedAssets()
    {
        // 标记需要卸载的资产
        List<string> toUnload = new List<string>();

        foreach (var kvp in loadedAssets)
        {
            bool inUse = activeLoads.Exists(request => request.assetPath == kvp.Key);

            if (!inUse)
            {
                toUnload.Add(kvp.Key);
            }
        }

        // 卸载
        foreach (string assetPath in toUnload)
        {
            Resources.UnloadAsset(loadedAssets[assetPath]);
            loadedAssets.Remove(assetPath);
        }

        // 调用Unity的垃圾回收
        Resources.UnloadUnusedAssets();
    }

    // 编辑器工具
    #if UNITY_EDITOR
    void OnDrawGizmosSelected()
    {
        // 可视化加载状态
        Gizmos.color = Color.blue;
        Gizmos.DrawWireSphere(transform.position, 10f);

        // 显示队列信息
        #if UNITY_EDITOR
        UnityEditor.Handles.Label(
            transform.position + Vector3.up * 2f,
            $"加载队列: {loadQueue.Count}\n活动加载: {activeLoads.Count}\n已缓存: {loadedAssets.Count}"
        );
        #endif
    }
    #endif
}
```

## 总结

环境与场景设计是游戏开发中创造沉浸式体验的核心环节。对于独立开发者而言，关键在于在有限资源下创造出令人信服、功能完善且性能优化的游戏世界。

**核心设计原则回顾**：

1. **功能优先**：环境设计首先服务于游戏玩法，其次才是视觉表现。
2. **模块化思维**：通过可重用组件构建多样化的环境，提高效率。
3. **氛围导向**：使用光照、色彩、天气等元素创造情感体验。
4. **优化先行**：从设计初期就考虑性能影响，避免后期重大修改。
5. **叙事整合**：让环境成为故事讲述者，而不仅仅是背景。

**给独立开发者的实用建议**：

1. **从小处开始**：先创建一个精心设计的小场景，验证工作流和风格。
2. **建立资产库**：投资时间创建高质量的模块化资产，长期受益。
3. **重用与变化**：通过纹理变化、组合方式和细节添加打破重复。
4. **测试在不同设备**：确保环境在目标平台上运行流畅。
5. **收集反馈**：让玩家测试环境，关注他们的导航体验和情感反应。

**技术趋势关注**：
1. **程序化生成**：使用算法辅助环境创建，提高效率和多样性。
2. **AI辅助设计**：AI工具在概念生成、纹理创建和布局优化中的应用。
3. **实时全局光照**：现代引擎的实时光照技术进步。
4. **虚拟制作**：电影制作技术向游戏开发的渗透。

记住：**最好的游戏环境不是最详细的，而是最能支持游戏体验的**。玩家记住的是在那个环境中经历的冒险，而不是环境本身的三角形数量。
