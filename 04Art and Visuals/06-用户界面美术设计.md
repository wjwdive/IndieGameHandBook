# 用户界面美术设计

## 概述

用户界面（UI）是玩家与游戏系统交互的桥梁，优秀的UI设计能够显著提升游戏的可玩性、可访问性和整体体验。UI美术不仅仅是视觉装饰，更是功能性的信息架构和交互设计。本章将深入探讨游戏UI美术设计的艺术与技术，特别关注独立游戏开发中的资源限制和实用解决方案，涵盖从概念设计到技术实现的完整流程。

**学习目标**：
- 理解游戏UI设计的核心原则和最佳实践
- 掌握UI美术设计的视觉语言和设计模式
- 学习UI动画和交互反馈的设计技巧
- 掌握多平台UI适配和响应式设计技术
- 了解UI性能优化和资源管理策略
- 能够为游戏项目制定完整的UI美术生产计划

## 1、UI设计基础

### 1.1 UI设计核心原则

**游戏UI的五个关键目标**：

1. **清晰性**：信息传达明确，无歧义
2. **可读性**：文字和图标在各种条件下易读
3. **一致性**：设计语言和交互模式统一
4. **响应性**：即时反馈玩家操作
5. **沉浸感**：与游戏世界风格协调，不破坏体验

**独立游戏UI设计理念**：
> **核心原则**：在资源有限的情况下，UI设计应遵循"形式追随功能"的原则。首先确保功能性完美，然后添加风格和美化。一个功能完善但外观简单的UI，远比一个华丽但难以使用的UI更好。

```python
def select_ui_style(game_type, resources, team_skills):
    """根据项目条件选择UI风格"""

    style_scores = {
        "diegetic": 0,
        "non_diegetic": 0,
        "spatial": 0,
        "meta": 0
    }

    # 根据游戏类型评分
    if game_type in ["horror", "immersive_sim", "narrative"]:
        style_scores["diegetic"] += 3
    elif game_type in ["rpg", "strategy", "simulation"]:
        style_scores["non_diegetic"] += 3
    elif game_type in ["sci_fi", "cyberpunk", "futuristic"]:
        style_scores["spatial"] += 3

    # 根据资源评分
    if resources == "low":
        style_scores["non_diegetic"] += 2  # 最简单实现
        style_scores["diegetic"] -= 1      # 需要额外资产
        style_scores["spatial"] -= 2       # 3D渲染复杂
    elif resources == "medium":
        style_scores["diegetic"] += 1
    elif resources == "high":
        style_scores["spatial"] += 2

    # 根据团队技能评分
    if "ui_design" in team_skills:
        style_scores["diegetic"] += 1
        style_scores["spatial"] += 1
    if "3d_modeling" in team_skills:
        style_scores["spatial"] += 2
    if "programming" in team_skills:
        style_scores["spatial"] += 1

    # 选择最高分
    selected_style = max(style_scores, key=style_scores.get)

    return {
        "selected_style": selected_style,
        "scores": style_scores,
        "rationale": f"基于游戏类型({game_type})、资源({resources})和团队技能"
    }
```

**色彩对比度检查**：
```python
def check_color_contrast(foreground, background):
    """检查色彩对比度是否符合WCAG标准"""

    # 计算相对亮度（sRGB）
    def luminance(color):
        r, g, b = color[0:2], color[2:4], color[4:6]
        r, g, b = int(r, 16)/255, int(g, 16)/255, int(b, 16)/255

        # sRGB转换
        r = r/12.92 if r <= 0.03928 else ((r+0.055)/1.055) ** 2.4
        g = g/12.92 if g <= 0.03928 else ((g+0.055)/1.055) ** 2.4
        b = b/12.92 if b <= 0.03928 else ((b+0.055)/1.055) ** 2.4

        return 0.2126 * r + 0.7152 * g + 0.0722 * b

    l1 = luminance(foreground)
    l2 = luminance(background)

    # 对比度比率
    contrast = (max(l1, l2) + 0.05) / (min(l1, l2) + 0.05)

    # WCAG标准
    standards = {
        "AA_normal_text": 4.5,     # 正常文本AA级
        "AA_large_text": 3.0,      # 大文本AA级
        "AAA_normal_text": 7.0,    # 正常文本AAA级
        "AAA_large_text": 4.5      # 大文本AAA级
    }

    results = {}
    for standard, threshold in standards.items():
        results[standard] = contrast >= threshold

    return {
        "contrast_ratio": round(contrast, 2),
        "wcag_results": results,
        "recommendations": generate_contrast_recommendations(contrast, results)
    }

def generate_contrast_recommendations(contrast, results):
    """生成对比度改进建议"""
    recommendations = []

    if not results["AA_normal_text"]:
        recommendations.append("文本对比度不足，建议调整颜色")
        if contrast < 3.0:
            recommendations.append("严重不足，考虑完全改变配色")
        elif contrast < 4.5:
            recommendations.append("接近标准，微调亮度或饱和度")

    return recommendations
```

```csharp
// Unity图标管理和使用系统
using UnityEngine;
using UnityEngine.UI;
using System.Collections.Generic;

public class IconSystem : MonoBehaviour
{
    [System.Serializable]
    public class IconSet
    {
        public string setName;
        public Sprite[] icons;
        public Dictionary<string, Sprite> iconDictionary;

        public void InitializeDictionary()
        {
            iconDictionary = new Dictionary<string, Sprite>();
            foreach (Sprite icon in icons)
            {
                if (icon != null)
                {
                    iconDictionary[icon.name] = icon;
                }
            }
        }

        public Sprite GetIcon(string iconName)
        {
            if (iconDictionary != null && iconDictionary.ContainsKey(iconName))
            {
                return iconDictionary[iconName];
            }
            return null;
        }
    }

    public IconSet[] iconSets;
    public Image[] dynamicIconContainers; // 需要动态更新图标的UI元素

    private Dictionary<string, IconSet> iconSetDictionary;

    void Start()
    {
        InitializeIconSystem();
    }

    void InitializeIconSystem()
    {
        iconSetDictionary = new Dictionary<string, IconSet>();

        foreach (IconSet set in iconSets)
        {
            set.InitializeDictionary();
            iconSetDictionary[set.setName] = set;
        }

        // 设置默认图标
        ApplyDefaultIcons();
    }

    public Sprite GetIcon(string setName, string iconName)
    {
        if (iconSetDictionary.ContainsKey(setName))
        {
            return iconSetDictionary[setName].GetIcon(iconName);
        }
        return null;
    }

    public void UpdateDynamicIcon(string containerName, string setName, string iconName)
    {
        Sprite icon = GetIcon(setName, iconName);
        if (icon == null)
        {
            Debug.LogWarning($"图标未找到: {setName}/{iconName}");
            return;
        }

        foreach (Image container in dynamicIconContainers)
        {
            if (container.gameObject.name == containerName)
            {
                container.sprite = icon;
                container.preserveAspect = true; // 保持比例
                break;
            }
        }
    }

    void ApplyDefaultIcons()
    {
        // 为标记了默认图标的容器设置图标
        foreach (Image container in dynamicIconContainers)
        {
            DefaultIcon defaultIcon = container.GetComponent<DefaultIcon>();
            if (defaultIcon != null)
            {
                Sprite icon = GetIcon(defaultIcon.iconSet, defaultIcon.iconName);
                if (icon != null)
                {
                    container.sprite = icon;
                }
            }
        }
    }

    // 编辑器工具
    #if UNITY_EDITOR
    [ContextMenu("更新所有图标引用")]
    void UpdateAllIconReferences()
    {
        // 在编辑器中更新所有图标引用
        Debug.Log("图标引用已更新");
    }
    #endif
}
```

```csharp
    public Color disabledColor = new Color(0.5f, 0.5f, 0.5f, 0.5f);
    public Color focusedColor = new Color(0.7f, 0.9f, 1f, 1f);

    [Header("动画设置")]
    public float hoverScale = 1.05f;
    public float pressedScale = 0.95f;
    public float animationDuration = 0.1f;
    public AnimationCurve animationCurve = AnimationCurve.EaseInOut(0, 0, 1, 1);

    [Header("声音设置")]
    public AudioClip hoverSound;
    public AudioClip clickSound;
    public AudioClip disabledSound;

    [Header("效果设置")]
    public GameObject hoverEffect;
    public GameObject clickEffect;

    private Image buttonImage;
    private Button buttonComponent;
    private RectTransform rectTransform;
    private AudioSource audioSource;

    private Vector3 originalScale;
    private Color originalColor;
    private Coroutine currentAnimation;

    void Awake()
    {
        buttonImage = GetComponent<Image>();
        buttonComponent = GetComponent<Button>();
        rectTransform = GetComponent<RectTransform>();
        audioSource = GetComponent<AudioSource>();

        if (audioSource == null)
        {
            audioSource = gameObject.AddComponent<AudioSource>();
            audioSource.playOnAwake = false;
        }

        originalScale = rectTransform.localScale;
        originalColor = buttonImage.color;
    }

    void Start()
    {
        // 初始状态
        UpdateVisualState();
    }

    void UpdateVisualState()
    {
        if (!buttonComponent.interactable)
        {
            buttonImage.color = disabledColor;
        }
        else
        {
            buttonImage.color = normalColor;
        }
    }

    // 鼠标悬停
    public void OnPointerEnter(PointerEventData eventData)
    {
        if (!buttonComponent.interactable) return;

        StartAnimation(hoverScale, hoverColor);

        if (hoverSound != null)
        {
            audioSource.PlayOneShot(hoverSound);
        }

        if (hoverEffect != null)
        {
            Instantiate(hoverEffect, transform.position, Quaternion.identity, transform);
        }
    }

    public void OnPointerExit(PointerEventData eventData)
    {
        if (!buttonComponent.interactable) return;

        StartAnimation(1f, normalColor);
    }

    // 鼠标按下/释放
    public void OnPointerDown(PointerEventData eventData)
    {
        if (!buttonComponent.interactable) return;

        StartAnimation(pressedScale, pressedColor);
    }

    public void OnPointerUp(PointerEventData eventData)
    {
        if (!buttonComponent.interactable) return;

        StartAnimation(hoverScale, hoverColor);
    }

    // 键盘导航
    public void OnSelect(BaseEventData eventData)
    {
        if (!buttonComponent.interactable) return;

        StartAnimation(hoverScale, focusedColor);
    }

    public void OnDeselect(BaseEventData eventData)
    {
        if (!buttonComponent.interactable) return;

        StartAnimation(1f, normalColor);
    }

    // 点击事件（附加到Button组件的OnClick）
    public void OnButtonClick()
    {
        if (!buttonComponent.interactable)
        {
            if (disabledSound != null)
            {
                audioSource.PlayOneShot(disabledSound);
            }
            return;
        }

        if (clickSound != null)
        {
            audioSource.PlayOneShot(clickSound);
        }

        if (clickEffect != null)
        {
            Instantiate(clickEffect, transform.position, Quaternion.identity);
        }

        // 点击动画
        StartCoroutine(ClickAnimation());
    }

    IEnumerator ClickAnimation()
    {
        // 快速缩放反馈
        rectTransform.localScale = originalScale * pressedScale;
        yield return new WaitForSeconds(0.1f);

        if (buttonComponent.interactable)
        {
            StartAnimation(hoverScale, hoverColor);
        }
        else
        {
            StartAnimation(1f, normalColor);
        }
    }

    void StartAnimation(float targetScale, Color targetColor)
    {
        if (currentAnimation != null)
        {
            StopCoroutine(currentAnimation);
        }

        currentAnimation = StartCoroutine(AnimateButton(targetScale, targetColor));
    }

    IEnumerator AnimateButton(float targetScale, Color targetColor)
    {
        Vector3 startScale = rectTransform.localScale;
        Vector3 endScale = originalScale * targetScale;

        Color startColor = buttonImage.color;

        float elapsedTime = 0f;

        while (elapsedTime < animationDuration)
        {
            elapsedTime += Time.deltaTime;
            float t = animationCurve.Evaluate(elapsedTime / animationDuration);

            rectTransform.localScale = Vector3.Lerp(startScale, endScale, t);
            buttonImage.color = Color.Lerp(startColor, targetColor, t);

            yield return null;
        }

        rectTransform.localScale = endScale;
        buttonImage.color = targetColor;

        currentAnimation = null;
    }

    // 外部控制
    public void SetInteractable(bool interactable)
    {
        buttonComponent.interactable = interactable;
        UpdateVisualState();
    }

    // 编辑器工具
    #if UNITY_EDITOR
    void Reset()
    {
        // 设置默认值
        normalColor = Color.white;
        hoverColor = new Color(0.9f, 0.9f, 0.9f, 1f);
        pressedColor = new Color(0.8f, 0.8f, 0.8f, 1f);
        disabledColor = new Color(0.5f, 0.5f, 0.5f, 0.5f);

        hoverScale = 1.05f;
        pressedScale = 0.95f;
        animationDuration = 0.1f;
    }
    #endif
}
```

```csharp
using UnityEngine;
using UnityEngine.UI;
using System.Collections;

public class AdvancedProgressBar : MonoBehaviour
{
    [Header("引用")]
    public Image fillImage;
    public Text progressText;
    public Image backgroundImage;
    public Image borderImage;

    [Header("进度设置")]
    [Range(0, 1)] public float currentProgress = 0f;
    [Range(0, 1)] public float targetProgress = 0f;
    public float maxValue = 100f;
    public float currentValue = 0f;

    [Header("动画设置")]
    public bool animateChanges = true;
    public float animationSpeed = 2f;
    public AnimationCurve animationCurve = AnimationCurve.EaseInOut(0, 0, 1, 1);

    [Header("视觉设置")]
    public Color fillColor = Color.green;
    public Color lowColor = Color.red;
    public Color mediumColor = Color.yellow;
    public Color highColor = Color.green;
    public bool useGradient = false;
    public Gradient progressGradient;

    [Header("文本设置")]
    public TextDisplayMode textMode = TextDisplayMode.Percentage;
    public string customFormat = "{0:0} / {1:0}";
    public bool showMaxValue = true;

    [Header("事件")]
    public UnityEventFloat onProgressChanged;
    public UnityEvent onProgressComplete;
    public UnityEventFloat onMilestoneReached;

    [System.Serializable]
    public class Milestone
    {
        public float threshold;
        public Color milestoneColor;
        public GameObject milestoneEffect;
        public string milestoneMessage;
        public bool hasTriggered = false;
    }

    public Milestone[] milestones;

    public enum TextDisplayMode
    {
        Percentage,
        Value,
        Custom,
        Hidden
    }

    private bool isAnimating = false;
    private Coroutine animationCoroutine;

    void Start()
    {
        UpdateProgressImmediate(currentProgress);
    }

    void Update()
    {
        // 测试控制
        if (Input.GetKeyDown(KeyCode.UpArrow))
        {
            SetProgress(currentProgress + 0.1f);
        }
        if (Input.GetKeyDown(KeyCode.DownArrow))
        {
            SetProgress(currentProgress - 0.1f);
        }
    }

    public void SetProgress(float progress)
    {
        progress = Mathf.Clamp01(progress);

        if (Mathf.Approximately(targetProgress, progress))
            return;

        targetProgress = progress;

        if (animateChanges && gameObject.activeInHierarchy)
        {
            if (animationCoroutine != null)
                StopCoroutine(animationCoroutine);

            animationCoroutine = StartCoroutine(AnimateProgress());
        }
        else
        {
            UpdateProgressImmediate(progress);
        }
    }

    public void SetValue(float value)
    {
        currentValue = Mathf.Clamp(value, 0, maxValue);
        SetProgress(currentValue / maxValue);
    }

    public void SetMaxValue(float newMaxValue)
    {
        maxValue = newMaxValue;
        UpdateProgressImmediate(currentProgress);
    }

    IEnumerator AnimateProgress()
    {
        isAnimating = true;
        float startProgress = currentProgress;
        float elapsedTime = 0f;

        while (elapsedTime < 1f / animationSpeed)
        {
            elapsedTime += Time.deltaTime;
            float t = animationCurve.Evaluate(elapsedTime * animationSpeed);

            currentProgress = Mathf.Lerp(startProgress, targetProgress, t);
            UpdateVisuals(currentProgress);

            yield return null;
        }

        currentProgress = targetProgress;
        UpdateVisuals(currentProgress);
        isAnimating = false;
        animationCoroutine = null;

        // 检查是否完成
        if (Mathf.Approximately(currentProgress, 1f))
        {
            onProgressComplete?.Invoke();
        }
    }

    void UpdateProgressImmediate(float progress)
    {
        currentProgress = progress;
        UpdateVisuals(progress);
    }

    void UpdateVisuals(float progress)
    {
        // 更新填充
        if (fillImage != null)
        {
            fillImage.fillAmount = progress;

            // 更新颜色
            if (useGradient && progressGradient != null)
            {
                fillImage.color = progressGradient.Evaluate(progress);
            }
            else
            {
                fillImage.color = GetProgressColor(progress);
            }
        }

        // 更新文本
        UpdateProgressText(progress);

        // 触发事件
        onProgressChanged?.Invoke(progress);

        // 检查里程碑
        CheckMilestones(progress);
    }

    Color GetProgressColor(float progress)
    {
        if (progress < 0.3f) return lowColor;
        if (progress < 0.7f) return mediumColor;
        return highColor;
    }

    void UpdateProgressText(float progress)
    {
        if (progressText == null) return;

        switch (textMode)
        {
            case TextDisplayMode.Percentage:
                progressText.text = $"{(progress * 100):0}%";
                break;
            case TextDisplayMode.Value:
                float current = progress * maxValue;
                if (showMaxValue)
                {
                    progressText.text = $"{current:0} / {maxValue:0}";
                }
                else
                {
                    progressText.text = $"{current:0}";
                }
                break;
            case TextDisplayMode.Custom:
                float currentValue = progress * maxValue;
                progressText.text = string.Format(customFormat, currentValue, maxValue);
                break;
            case TextDisplayMode.Hidden:
                progressText.text = "";
                break;
        }
    }

    void CheckMilestones(float progress)
    {
        if (milestones == null) return;

        foreach (Milestone milestone in milestones)
        {
            if (!milestone.hasTriggered && progress >= milestone.threshold)
            {
                milestone.hasTriggered = true;

                // 颜色变化
                if (milestone.milestoneColor != Color.clear && !useGradient)
                {
                    fillImage.color = milestone.milestoneColor;
                }

                // 特效
                if (milestone.milestoneEffect != null)
                {
                    Instantiate(milestone.milestoneEffect, transform.position, Quaternion.identity);
                }

                // 消息
                if (!string.IsNullOrEmpty(milestone.milestoneMessage))
                {
                    Debug.Log($"里程碑达成: {milestone.milestoneMessage}");
                }

                // 事件
                onMilestoneReached?.Invoke(milestone.threshold);
            }
        }
    }

    public void ResetMilestones()
    {
        if (milestones == null) return;

        foreach (Milestone milestone in milestones)
        {
            milestone.hasTriggered = false;
        }
    }

    public void ResetProgress()
    {
        SetProgress(0f);
        ResetMilestones();
    }

    // 编辑器工具
    #if UNITY_EDITOR
    void OnValidate()
    {
        if (!Application.isPlaying)
        {
            UpdateProgressImmediate(currentProgress);
        }
    }
```

```csharp
// Unity多设备菜单导航系统
using UnityEngine;
using UnityEngine.EventSystems;
using UnityEngine.UI;
using System.Collections.Generic;

public class UniversalMenuNavigator : MonoBehaviour
{
    [Header("导航设置")]
    public NavigationMode navigationMode = NavigationMode.Auto;
    public float gamepadDeadzone = 0.5f;
    public float navigationRepeatDelay = 0.5f;
    public float navigationRepeatRate = 0.1f;

    [Header("视觉反馈")]
    public Color normalColor = Color.white;
    public Color focusedColor = Color.yellow;
    public Color selectedColor = Color.green;
    public GameObject focusIndicator;

    [Header("音频反馈")]
    public AudioClip navigationSound;
    public AudioClip selectionSound;
    public AudioClip errorSound;

    private EventSystem eventSystem;
    private StandaloneInputModule inputModule;
    private AudioSource audioSource;

    private GameObject lastSelectedObject;
    private Dictionary<GameObject, MenuItem> menuItems = new Dictionary<GameObject, MenuItem>();

    private float lastNavigationTime = 0f;
    private Vector2 lastGamepadInput = Vector2.zero;

    public enum NavigationMode
    {
        Auto,           // 自动检测
        MouseKeyboard,  // 鼠标键盘
        Gamepad,        // 游戏手柄
        Touch           // 触摸屏
    }

    [System.Serializable]
    public class MenuItem
    {
        public GameObject gameObject;
        public Selectable selectable;
        public Image highlightImage;
        public Button button;
        public Navigation navigation;

        public MenuItem(GameObject go)
        {
            gameObject = go;
            selectable = go.GetComponent<Selectable>();
            highlightImage = go.GetComponent<Image>();
            button = go.GetComponent<Button>();

            if (selectable != null)
            {
                navigation = selectable.navigation;
            }
        }

        public void SetHighlight(bool highlight, Color color)
        {
            if (highlightImage != null)
            {
                highlightImage.color = highlight ? color : Color.white;
            }
        }
    }

    void Start()
    {
        eventSystem = FindObjectOfType<EventSystem>();
        if (eventSystem == null)
        {
            GameObject es = new GameObject("EventSystem");
            eventSystem = es.AddComponent<EventSystem>();
            es.AddComponent<StandaloneInputModule>();
        }

        inputModule = eventSystem.GetComponent<StandaloneInputModule>();
        audioSource = GetComponent<AudioSource>();

        if (audioSource == null)
        {
            audioSource = gameObject.AddComponent<AudioSource>();
            audioSource.playOnAwake = false;
        }

        // 收集所有菜单项
        CollectMenuItems();

        // 设置初始选择
        SetInitialSelection();

        // 根据模式配置
        ConfigureForNavigationMode();
    }

    void Update()
    {
        // 检测输入设备变化
        DetectInputDeviceChange();

        // 处理游戏手柄导航
        if (navigationMode == NavigationMode.Gamepad || navigationMode == NavigationMode.Auto)
        {
            HandleGamepadNavigation();
        }

        // 更新焦点指示器
        UpdateFocusIndicator();

        // 记录最后选择的对象
        if (eventSystem.currentSelectedGameObject != null &&
            eventSystem.currentSelectedGameObject != lastSelectedObject)
        {
            OnSelectionChanged(lastSelectedObject, eventSystem.currentSelectedGameObject);
            lastSelectedObject = eventSystem.currentSelectedGameObject;
        }
    }

    void CollectMenuItems()
    {
        // 查找所有可选择的UI元素
        Selectable[] allSelectables = FindObjectsOfType<Selectable>();

        foreach (Selectable selectable in allSelectables)
        {
            if (selectable.gameObject.activeInHierarchy && selectable.interactable)
            {
                MenuItem item = new MenuItem(selectable.gameObject);
                menuItems[selectable.gameObject] = item;
            }
        }
    }

    void SetInitialSelection()
    {
        // 查找第一个可选择的项目
        foreach (var kvp in menuItems)
        {
            if (kvp.Value.selectable != null && kvp.Value.selectable.interactable)
            {
                eventSystem.SetSelectedGameObject(kvp.Key);
                lastSelectedObject = kvp.Key;
                break;
            }
        }
    }

    void ConfigureForNavigationMode()
    {
        switch (navigationMode)
        {
            case NavigationMode.MouseKeyboard:
                // 启用鼠标输入
                if (inputModule != null)
                {
                    inputModule.forceModuleActive = true;
                }
                Cursor.visible = true;
                Cursor.lockState = CursorLockMode.None;
                break;

            case NavigationMode.Gamepad:
                // 禁用鼠标，启用游戏手柄
                if (inputModule != null)
                {
                    inputModule.forceModuleActive = false;
                }
                Cursor.visible = false;
                Cursor.lockState = CursorLockMode.Locked;
                break;

            case NavigationMode.Touch:
                // 触摸屏配置
                Cursor.visible = false;
                break;
        }
    }

    void DetectInputDeviceChange()
    {
        if (navigationMode != NavigationMode.Auto) return;

        // 检测鼠标移动
        if (Input.GetAxis("Mouse X") != 0 || Input.GetAxis("Mouse Y") != 0)
        {
            SetNavigationMode(NavigationMode.MouseKeyboard);
            return;
        }

        // 检测游戏手柄输入
        if (Mathf.Abs(Input.GetAxis("Horizontal")) > gamepadDeadzone ||
            Mathf.Abs(Input.GetAxis("Vertical")) > gamepadDeadzone ||
            Input.GetButtonDown("Submit") || Input.GetButtonDown("Cancel"))
        {
            SetNavigationMode(NavigationMode.Gamepad);
            return;
        }

        // 检测触摸
        if (Input.touchCount > 0)
        {
            SetNavigationMode(NavigationMode.Touch);
        }
    }

    void SetNavigationMode(NavigationMode newMode)
    {
        if (navigationMode == newMode) return;

        navigationMode = newMode;
        ConfigureForNavigationMode();

        Debug.Log($"导航模式切换为: {newMode}");
    }

    void HandleGamepadNavigation()
    {
        Vector2 input = new Vector2(
            Input.GetAxis("Horizontal"),
            Input.GetAxis("Vertical")
        );

        // 应用死区
        if (input.magnitude < gamepadDeadzone)
        {
            lastGamepadInput = Vector2.zero;
            return;
        }

        // 检查重复延迟
        if (Time.time - lastNavigationTime < navigationRepeatDelay &&
            Vector2.Dot(input, lastGamepadInput) > 0.8f)
        {
            return;
        }

        // 导航
        GameObject current = eventSystem.currentSelectedGameObject;
        if (current != null && menuItems.ContainsKey(current))
        {
            Selectable selectable = menuItems[current].selectable;
            Selectable next = null;

            if (Mathf.Abs(input.x) > Mathf.Abs(input.y))
            {
                // 水平导航
                if (input.x > 0)
                {
                    next = selectable.FindSelectableOnRight();
                }
                else
                {
                    next = selectable.FindSelectableOnLeft();
                }
            }
            else
            {
                // 垂直导航
                if (input.y > 0)
                {
                    next = selectable.FindSelectableOnUp();
                }
                else
                {
                    next = selectable.FindSelectableOnDown();
                }
            }

            if (next != null && next.gameObject != current)
            {
                eventSystem.SetSelectedGameObject(next.gameObject);
                PlayNavigationSound();
                lastNavigationTime = Time.time;
            }
        }

        lastGamepadInput = input;
    }

    void UpdateFocusIndicator()
    {
        if (focusIndicator == null) return;

        GameObject current = eventSystem.currentSelectedGameObject;
        if (current != null)
        {
            focusIndicator.SetActive(true);

            RectTransform currentRect = current.GetComponent<RectTransform>();
            RectTransform indicatorRect = focusIndicator.GetComponent<RectTransform>();

            // 定位到当前选择的对象
            indicatorRect.position = currentRect.position;
            indicatorRect.sizeDelta = currentRect.sizeDelta + new Vector2(10, 10);
        }
        else
        {
            focusIndicator.SetActive(false);
        }
    }

    void OnSelectionChanged(GameObject previous, GameObject current)
    {
        // 更新视觉反馈
        if (previous != null && menuItems.ContainsKey(previous))
        {
            menuItems[previous].SetHighlight(false, normalColor);
        }

        if (current != null && menuItems.ContainsKey(current))
        {
            menuItems[current].SetHighlight(true, focusedColor);
        }

        // 播放声音
        PlayNavigationSound();
    }

    void PlayNavigationSound()
    {
        if (navigationSound != null && audioSource != null)
        {
            audioSource.PlayOneShot(navigationSound);
        }
    }

    public void PlaySelectionSound()
    {
        if (selectionSound != null && audioSource != null)
        {
            audioSource.PlayOneShot(selectionSound);
        }
    }

    public void PlayErrorSound()
    {
        if (errorSound != null && audioSource != null)
        {
            audioSource.PlayOneShot(errorSound);
        }
    }

    // 公共方法
    public void SelectItem(GameObject item)
    {
        if (item != null && menuItems.ContainsKey(item))
        {
            eventSystem.SetSelectedGameObject(item);
        }
    }

    public void EnableNavigation(bool enable)
    {
        foreach (var kvp in menuItems)
        {

            {
                kvp.Value.selectable.interactable = enable;
            }

        {
            eventSystem.SetSelectedGameObject(null);
        }
    }
```

```csharp
// Unity UI动画控制器
using UnityEngine;
using UnityEngine.UI;
using System.Collections;
using System.Collections.Generic;

[RequireComponent(typeof(CanvasGroup))]
public class UIAnimationController : MonoBehaviour
{
    [Header("入场动画")]
    public AnimationType entranceAnimation = AnimationType.FadeIn;
    public float entranceDuration = 0.3f;
    public AnimationCurve entranceCurve = AnimationCurve.EaseInOut(0, 0, 1, 1);
    public bool playOnEnable = true;

    [Header("退场动画")]
    public AnimationType exitAnimation = AnimationType.FadeOut;
    public float exitDuration = 0.2f;
    public AnimationCurve exitCurve = AnimationCurve.EaseInOut(0, 0, 1, 1);

    [Header("交互动画")]
    public bool hoverAnimation = true;
    public float hoverScaleAmount = 1.1f;
    public float hoverAnimationDuration = 0.15f;

    [Header("注意力动画")]
    public bool attentionAnimation = false;
    public AttentionType attentionType = AttentionType.Pulse;
    public float attentionInterval = 2f;
    public float attentionIntensity = 1.2f;

    private CanvasGroup canvasGroup;
    private RectTransform rectTransform;
    private Vector3 originalScale;
    private Color originalColor;

    private Coroutine currentAnimation;
    private bool isAnimating = false;

    public enum AnimationType
    {
        None,
        FadeIn,
        FadeOut,
        SlideInFromLeft,
        SlideInFromRight,
        SlideInFromTop,
        SlideInFromBottom,
        SlideOutToLeft,
        SlideOutToRight,
        SlideOutToTop,
        SlideOutToBottom,
        ScaleIn,
        ScaleOut,
        RotateIn,
        RotateOut
    }

    public enum AttentionType
    {
        Pulse,      // 脉动
        Shake,      // 抖动
        Bounce,     // 弹跳
        Highlight   // 高亮
    }

    void Awake()
    {
        canvasGroup = GetComponent<CanvasGroup>();
        rectTransform = GetComponent<RectTransform>();

        originalScale = rectTransform.localScale;

        Image image = GetComponent<Image>();
        if (image != null)
        {
            originalColor = image.color;
        }
    }

    void OnEnable()
    {
        if (playOnEnable)
        {
            PlayEntranceAnimation();
        }

        if (attentionAnimation)
        {
            StartCoroutine(AttentionAnimationRoutine());
        }
    }

    void OnDisable()
    {
        StopAllCoroutines();
        isAnimating = false;
        currentAnimation = null;
    }

    public void PlayEntranceAnimation()
    {
        if (isAnimating && currentAnimation != null)
        {
            StopCoroutine(currentAnimation);
        }

        currentAnimation = StartCoroutine(PlayAnimation(entranceAnimation, entranceDuration, entranceCurve, true));
    }

    public void PlayExitAnimation(System.Action onComplete = null)
    {
        if (isAnimating && currentAnimation != null)
        {
            StopCoroutine(currentAnimation);
        }

        currentAnimation = StartCoroutine(PlayAnimation(exitAnimation, exitDuration, exitCurve, false, onComplete));
    }

    IEnumerator PlayAnimation(AnimationType type, float duration, AnimationCurve curve, bool isEntrance, System.Action onComplete = null)
    {
        isAnimating = true;

        // 初始状态
        Vector3 startPosition = rectTransform.anchoredPosition;
        Vector3 startScale = rectTransform.localScale;
        float startAlpha = canvasGroup.alpha;
        float startRotation = rectTransform.localEulerAngles.z;

        // 目标状态
        Vector3 targetPosition = startPosition;
        Vector3 targetScale = startScale;
        float targetAlpha = startAlpha;
        float targetRotation = startRotation;

        // 根据动画类型设置目标
        SetAnimationTargets(type, ref targetPosition, ref targetScale, ref targetAlpha, ref targetRotation, isEntrance);

        float elapsedTime = 0f;

        while (elapsedTime < duration)
        {
            elapsedTime += Time.deltaTime;
            float t = curve.Evaluate(elapsedTime / duration);

            // 插值所有属性
            if (type.ToString().Contains("Slide") || type.ToString().Contains("Scale"))
            {
                rectTransform.anchoredPosition = Vector3.Lerp(startPosition, targetPosition, t);
                rectTransform.localScale = Vector3.Lerp(startScale, targetScale, t);
            }

            if (type.ToString().Contains("Fade"))
            {
                canvasGroup.alpha = Mathf.Lerp(startAlpha, targetAlpha, t);
            }

            if (type.ToString().Contains("Rotate"))
            {
                Vector3 rotation = rectTransform.localEulerAngles;
                rotation.z = Mathf.LerpAngle(startRotation, targetRotation, t);
                rectTransform.localEulerAngles = rotation;
            }

            yield return null;
        }

        // 确保到达目标状态
        rectTransform.anchoredPosition = targetPosition;
        rectTransform.localScale = targetScale;
        canvasGroup.alpha = targetAlpha;

        if (type.ToString().Contains("Rotate"))
        {
            Vector3 rotation = rectTransform.localEulerAngles;
            rotation.z = targetRotation;
            rectTransform.localEulerAngles = rotation;
        }

        isAnimating = false;
        currentAnimation = null;

        onComplete?.Invoke();
    }

    void SetAnimationTargets(AnimationType type, ref Vector3 position, ref Vector3 scale, ref float alpha, ref float rotation, bool isEntrance)
    {
        switch (type)
        {
            case AnimationType.FadeIn:
                alpha = isEntrance ? 1f : 0f;
                break;

            case AnimationType.FadeOut:
                alpha = isEntrance ? 0f : 1f;
                break;

            case AnimationType.SlideInFromLeft:
                position.x = isEntrance ? 0f : -Screen.width;
                break;

            case AnimationType.SlideInFromRight:
                position.x = isEntrance ? 0f : Screen.width;
                break;

            case AnimationType.SlideInFromTop:
                position.y = isEntrance ? 0f : Screen.height;
                break;

            case AnimationType.SlideInFromBottom:
                position.y = isEntrance ? 0f : -Screen.height;
                break;

            case AnimationType.ScaleIn:
                scale = isEntrance ? originalScale : Vector3.zero;
                break;

            case AnimationType.ScaleOut:
                scale = isEntrance ? Vector3.zero : originalScale;
                break;

            case AnimationType.RotateIn:
                rotation = isEntrance ? 0f : -180f;
                break;

            case AnimationType.RotateOut:
                rotation = isEntrance ? 180f : 0f;
                break;
        }
    }

    // 悬停动画
    public void OnPointerEnter()
    {
        if (!hoverAnimation || isAnimating) return;

        StartCoroutine(ScaleAnimation(originalScale * hoverScaleAmount, hoverAnimationDuration));
    }

    public void OnPointerExit()
    {

    }

    IEnumerator ScaleAnimation(Vector3 targetScale, float duration)
    {
        Vector3 startScale = rectTransform.localScale;
        float elapsedTime = 0f;

        while (elapsedTime < duration)
        {
            elapsedTime += Time.deltaTime;
            float t = elapsedTime / duration;

            rectTransform.localScale = Vector3.Lerp(startScale, targetScale, t);

            yield return null;
        }

        rectTransform.localScale = targetScale;
    }

    // 注意力动画
    IEnumerator AttentionAnimationRoutine()
    {
        while (attentionAnimation && gameObject.activeInHierarchy)
        {
            yield return new WaitForSeconds(attentionInterval);

            switch (attentionType)
            {
                case AttentionType.Pulse:
                    yield return StartCoroutine(PulseAnimation());
                    break;

                case AttentionType.Shake:
                    yield return StartCoroutine(ShakeAnimation());
                    break;

                case AttentionType.Bounce:
                    yield return StartCoroutine(BounceAnimation());
                    break;

                case AttentionType.Highlight:
                    yield return StartCoroutine(HighlightAnimation());
                    break;
            }
        }
    }

    IEnumerator PulseAnimation()
    {
        Vector3 pulseScale = originalScale * attentionIntensity;
        yield return StartCoroutine(ScaleAnimation(pulseScale, 0.2f));
        yield return StartCoroutine(ScaleAnimation(originalScale, 0.2f));
    }

    IEnumerator ShakeAnimation()
    {
        Vector3 originalPos = rectTransform.anchoredPosition;
        float shakeAmount = 10f;

        for (int i = 0; i < 3; i++)
        {
            rectTransform.anchoredPosition = originalPos + new Vector3(shakeAmount, 0, 0);
            yield return new WaitForSeconds(0.05f);
            rectTransform.anchoredPosition = originalPos - new Vector3(shakeAmount, 0, 0);
            yield return new WaitForSeconds(0.05f);
        }

        rectTransform.anchoredPosition = originalPos;
    }

    IEnumerator BounceAnimation()
    {
        float bounceHeight = 20f;
        Vector3 originalPos = rectTransform.anchoredPosition;

        // 向上
        rectTransform.anchoredPosition = originalPos + new Vector3(0, bounceHeight, 0);
        yield return new WaitForSeconds(0.1f);

        // 向下（稍微超过）
        rectTransform.anchoredPosition = originalPos - new Vector3(0, bounceHeight * 0.5f, 0);
        yield return new WaitForSeconds(0.1f);

        // 回到原位
        rectTransform.anchoredPosition = originalPos;
    }

    IEnumerator HighlightAnimation()
    {
        Image image = GetComponent<Image>();
        if (image == null) yield break;

        Color highlightColor = originalColor * attentionIntensity;
        image.color = highlightColor;
        yield return new WaitForSeconds(0.3f);
        image.color = originalColor;
    }

    // 公共控制方法
    public void SetAttentionAnimation(bool enabled)
    {
        attentionAnimation = enabled;

        if (enabled && gameObject.activeInHierarchy)
        {
            StartCoroutine(AttentionAnimationRoutine());
        }
    }

    public void StopCurrentAnimation()
    {
        if (currentAnimation != null)
        {
            StopCoroutine(currentAnimation);
            isAnimating = false;
            currentAnimation = null;
        }
    }

    // 编辑器工具
    #if UNITY_EDITOR

        PlayEntranceAnimation();
```

```csharp
// Unity多平台触觉反馈
using UnityEngine;
using System.Collections;
#if UNITY_IOS
using UnityEngine.iOS;
#endif
#if UNITY_ANDROID
using UnityEngine.Android;
#endif

public class HapticFeedbackSystem : MonoBehaviour
{
    [System.Serializable]
    public class HapticPreset
    {
        public string presetName;
        public float intensity = 0.5f;
        public float duration = 0.1f;
        public HapticPattern pattern = HapticPattern.Single;

        // iOS特定
        public iOSHapticType iOSType = iOSHapticType.Success;

        // Android特定
        public long androidDuration = 50; // 毫秒
        public int androidAmplitude = 100; // 0-255
    }

    public enum HapticPattern
    {
        Single,         // 单次震动
        Double,         // 两次快速震动
        Triple,         // 三次快速震动
        Long,           // 长震动
        Pulse,          // 脉动震动
        Custom          // 自定义模式
    }

    public enum iOSHapticType
    {
        Selection,
        Success,
        Warning,
        Error,
        LightImpact,
        MediumImpact,
        HeavyImpact,
        RigidImpact,
        SoftImpact
    }

    [Header("预设")]
    public HapticPreset[] hapticPresets;
    public bool hapticsEnabled = true;

    [Header("平台设置")]
    public bool useNativeHaptics = true;
    public float fallbackVibrationDuration = 0.1f;

    private Dictionary<string, HapticPreset> presetDictionary;
    private Coroutine vibrationCoroutine;

    void Start()
    {
        InitializeHapticSystem();
    }

    void InitializeHapticSystem()
    {
        presetDictionary = new Dictionary<string, HapticPreset>();

        foreach (HapticPreset preset in hapticPresets)
        {
            presetDictionary[preset.presetName] = preset;
        }

        // 检查平台支持
        CheckPlatformSupport();
    }

    void CheckPlatformSupport()
    {
        #if UNITY_IOS
            if (Device.generation >= DeviceGeneration.iPhone6S)
            {
                Debug.Log("iOS设备支持触觉反馈");
            }
            else
            {
                Debug.Log("iOS设备不支持高级触觉反馈，将使用回退方案");
                useNativeHaptics = false;
            }
        #elif UNITY_ANDROID
            // Android设备通常支持震动，但质量参差不齐
            Debug.Log("Android设备支持基础震动");
        #else
            Debug.Log("当前平台可能不支持触觉反馈");
            useNativeHaptics = false;
        #endif
    }

    public void TriggerHaptic(string presetName)
    {
        if (!hapticsEnabled) return;

        if (presetDictionary.ContainsKey(presetName))
        {
            HapticPreset preset = presetDictionary[presetName];
            TriggerHaptic(preset);
        }
        else
        {
            Debug.LogWarning($"触觉预设未找到: {presetName}");
            TriggerDefaultHaptic();
        }
    }

    public void TriggerHaptic(HapticPreset preset)
    {
        if (!hapticsEnabled) return;

        if (useNativeHaptics)
        {
            TriggerNativeHaptic(preset);
        }
        else
        {
            TriggerFallbackHaptic(preset);
        }
    }

    void TriggerNativeHaptic(HapticPreset preset)
    {
        #if UNITY_IOS
            TriggeriOSHaptic(preset.iOSType);
        #elif UNITY_ANDROID
            TriggerAndroidHaptic(preset.androidDuration, preset.androidAmplitude);
        #else
            // 其他平台使用回退
            TriggerFallbackHaptic(preset);
        #endif
    }

    #if UNITY_IOS
    void TriggeriOSHaptic(iOSHapticType type)
    {
        if (!hapticsEnabled) return;

        try
        {
            switch (type)
            {
                case iOSHapticType.Selection:
                    // 轻触反馈
                    if (System.Enum.IsDefined(typeof(iOSHapticType), "Selection"))
                    {
                        // 使用iOS原生API
                        // 注意：这需要iOS 10+
                    }
                    break;

                case iOSHapticType.Success:
                    // 成功反馈
                    break;

                case iOSHapticType.Warning:
                    // 警告反馈
                    break;

                case iOSHapticType.Error:
                    // 错误反馈
                    break;

                case iOSHapticType.LightImpact:
                case iOSHapticType.MediumImpact:
                case iOSHapticType.HeavyImpact:
                case iOSHapticType.RigidImpact:
                case iOSHapticType.SoftImpact:
                    // 冲击反馈
                    break;
            }
        }
        catch (System.Exception e)
        {
            Debug.LogError($"iOS触觉反馈错误: {e.Message}");
            TriggerFallbackHaptic(new HapticPreset { intensity = 0.5f, duration = 0.1f });
        }
    }
    #endif

    #if UNITY_ANDROID
    void TriggerAndroidHaptic(long duration, int amplitude)
    {
        if (!hapticsEnabled) return;

        try
        {
            // 使用Android震动API
            // 注意：需要VIBRATE权限
            if (Permission.HasUserAuthorizedPermission("android.permission.VIBRATE"))
            {
                // Android API调用
                // 简化实现，实际需要调用Android Java类
                Handheld.Vibrate();

            {
                // 请求权限
                Permission.RequestUserPermission("android.permission.VIBRATE");
                Debug.Log("需要震动权限");
            }
        }
        catch (System.Exception e)
        {
            Debug.LogError($"Android触觉反馈错误: {e.Message}");
            TriggerFallbackHaptic(new HapticPreset { intensity = 0.5f, duration = 0.1f });
        }
    }
    #endif

    void TriggerFallbackHaptic(HapticPreset preset)
    {
        if (!hapticsEnabled) return;

        // 使用Unity的手持设备震动作为回退
        if (SystemInfo.supportsVibration)
        {
            // 根据模式触发不同震动
            switch (preset.pattern)
            {
                case HapticPattern.Single:
                    Handheld.Vibrate();
                    break;

                case HapticPattern.Double:
                    if (vibrationCoroutine != null)
                        StopCoroutine(vibrationCoroutine);
                    vibrationCoroutine = StartCoroutine(VibrationPattern(0.05f, 0.05f, 0.05f));
                    break;

                case HapticPattern.Triple:
                    if (vibrationCoroutine != null)
                        StopCoroutine(vibrationCoroutine);
                    vibrationCoroutine = StartCoroutine(VibrationPattern(0.05f, 0.05f, 0.05f, 0.05f, 0.05f));
                    break;

                case HapticPattern.Long:
                    Handheld.Vibrate();
                    break;

                case HapticPattern.Pulse:
                    if (vibrationCoroutine != null)
                        StopCoroutine(vibrationCoroutine);
                    vibrationCoroutine = StartCoroutine(PulseVibration(0.1f, 3, 0.1f));
                    break;

                case HapticPattern.Custom:
                    // 自定义模式需要额外参数
                    Handheld.Vibrate();
                    break;
            }
        }
        else
        {
            Debug.Log("设备不支持震动");
        }
    }

    IEnumerator VibrationPattern(params float[] pattern)
    {
        for (int i = 0; i < pattern.Length; i += 2)
        {
            if (i < pattern.Length - 1)
            {
                // 震动
                Handheld.Vibrate();
                yield return new WaitForSeconds(pattern[i]);

                // 暂停
                yield return new WaitForSeconds(pattern[i + 1]);
            }
        }
    }

    IEnumerator PulseVibration(float pulseDuration, int pulseCount, float interval)
    {
        for (int i = 0; i < pulseCount; i++)
        {
            Handheld.Vibrate();
            yield return new WaitForSeconds(pulseDuration);

            if (i < pulseCount - 1)
            {
                yield return new WaitForSeconds(interval);
            }
        }
    }

    void TriggerDefaultHaptic()
    {
        HapticPreset defaultPreset = new HapticPreset
        {
            presetName = "default",
            intensity = 0.5f,
            duration = 0.1f,
            pattern = HapticPattern.Single
        };

        TriggerHaptic(defaultPreset);
    }

    // 预设快捷方法
    public void SuccessFeedback()
    {
        TriggerHaptic("success");
    }

    public void ErrorFeedback()
    {
        TriggerHaptic("error");
    }

    public void SelectionFeedback()
    {
        TriggerHaptic("selection");
    }

    public void ImpactFeedback(float intensity = 0.5f)
    {
```

```csharp
// Unity UI性能分析器
using UnityEngine;
using UnityEngine.UI;
using System.Collections.Generic;
using System.Text;

public class UIPerformanceAnalyzer : MonoBehaviour
{
    [Header("分析设置")]
    public bool autoAnalyzeOnStart = true;
    public float analysisInterval = 5f;
    public bool showInGameDisplay = false;

    [Header("性能阈值")]
    public int maxDrawCalls = 50;
    public int maxCanvasCount = 5;
    public int maxActiveElements = 100;
    public float maxRebuildTime = 5f; // 毫秒

    private Canvas[] allCanvases;
    private List<PerformanceIssue> performanceIssues = new List<PerformanceIssue>();
    private float lastAnalysisTime = 0f;
    private StringBuilder reportBuilder = new StringBuilder();

    [System.Serializable]
    public class PerformanceIssue
    {
        public string issueType;
        public string description;
        public string location;
        public string recommendation;
        public SeverityLevel severity;

        public enum SeverityLevel
        {
            Low,     // 轻微问题
            Medium,  // 中等问题
            High,    // 严重问题
            Critical // 关键问题
        }
    }

    void Start()
    {
        if (autoAnalyzeOnStart)
        {
            AnalyzeAllUI();
        }
    }

    void Update()
    {
        if (Time.time - lastAnalysisTime >= analysisInterval)
        {
            AnalyzeAllUI();
            lastAnalysisTime = Time.time;
        }

        if (showInGameDisplay && Input.GetKeyDown(KeyCode.F12))
        {
            TogglePerformanceDisplay();
        }
    }

    public void AnalyzeAllUI()
    {
        performanceIssues.Clear();
        reportBuilder.Clear();

        allCanvases = FindObjectsOfType<Canvas>();

        // 各项分析
        AnalyzeCanvasCount();
        AnalyzeDrawCalls();
        AnalyzeActiveElements();
        AnalyzeLayoutRebuild();
        AnalyzeTextureMemory();
        AnalyzeRaycastTargets();

        // 生成报告
        GenerateReport();

        // 显示警告
        DisplayWarnings();
    }

    void AnalyzeCanvasCount()
    {
        int canvasCount = allCanvases.Length;

        if (canvasCount > maxCanvasCount)
        {
            performanceIssues.Add(new PerformanceIssue
            {
                issueType = "Canvas数量过多",
                description = $"发现{canvasCount}个Canvas，建议合并",
                location = "场景中所有Canvas",
                recommendation = "合并相关UI到同一个Canvas，使用Canvas Group控制显示",
                severity = canvasCount > 10 ? PerformanceIssue.SeverityLevel.High : PerformanceIssue.SeverityLevel.Medium
            });
        }
    }

    void AnalyzeDrawCalls()
    {
        // 简化：统计使用不同材质的UI元素
        int uniqueMaterials = 0;
        HashSet<Material> materialSet = new HashSet<Material>();

        foreach (Canvas canvas in allCanvases)
        {
            Graphic[] graphics = canvas.GetComponentsInChildren<Graphic>(true);
            foreach (Graphic graphic in graphics)
            {
                if (graphic.material != null && !materialSet.Contains(graphic.material))
                {
                    materialSet.Add(graphic.material);
                    uniqueMaterials++;
                }
            }
        }

        if (uniqueMaterials > maxDrawCalls / 2) // 粗略估计
        {
            performanceIssues.Add(new PerformanceIssue
            {
                issueType = "潜在绘制调用过多",
                description = $"发现{uniqueMaterials}种不同材质，可能导致高绘制调用",
                location = "多个UI元素",
                recommendation = "使用纹理图集，合并相似材质",
                severity = uniqueMaterials > maxDrawCalls ? PerformanceIssue.SeverityLevel.High : PerformanceIssue.SeverityLevel.Medium
            });
        }
    }

    void AnalyzeActiveElements()
    {
        int totalActiveElements = 0;

        foreach (Canvas canvas in allCanvases)
        {
            Graphic[] graphics = canvas.GetComponentsInChildren<Graphic>(true);
            totalActiveElements += graphics.Length;
        }

        if (totalActiveElements > maxActiveElements)
        {
            performanceIssues.Add(new PerformanceIssue
            {
                issueType = "活动UI元素过多",
                description = $"发现{totalActiveElements}个活动UI元素",
                location = "多个Canvas",
                recommendation = "禁用不可见UI，使用对象池，优化UI结构",
                severity = PerformanceIssue.SeverityLevel.Medium
            });
        }
    }

    void AnalyzeLayoutRebuild()
    {
        // 简化分析：查找频繁变化的布局元素
        // 实际项目中可能需要更复杂的分析

        performanceIssues.Add(new PerformanceIssue
            {
                issueType = "布局重建分析",
                description = "需要运行时分析布局重建频率",
                location = "动态UI元素",
                recommendation = "使用Profiler分析Layout重建，避免频繁变化",
                severity = PerformanceIssue.SeverityLevel.Low
            });
    }

    void AnalyzeTextureMemory()
    {
        long totalTextureMemory = 0;
        HashSet<Texture> textureSet = new HashSet<Texture>();

        foreach (Canvas canvas in allCanvases)
        {
            Graphic[] graphics = canvas.GetComponentsInChildren<Graphic>(true);
            foreach (Graphic graphic in graphics)
            {
                if (graphic.mainTexture != null && !textureSet.Contains(graphic.mainTexture))
                {
                    textureSet.Add(graphic.mainTexture);
                    totalTextureMemory += EstimateTextureMemory(graphic.mainTexture);
                }
            }
        }

        // 转换为MB
        float memoryMB = totalTextureMemory / (1024f * 1024f);

        if (memoryMB > 50) // 50MB阈值
        {
            performanceIssues.Add(new PerformanceIssue
            {
                issueType = "UI纹理内存过高",
                description = $"UI纹理占用约{memoryMB:F1}MB内存",
                location = "多个UI纹理",
                recommendation = "压缩纹理，使用合适格式，使用纹理图集",
                severity = memoryMB > 100 ? PerformanceIssue.SeverityLevel.High : PerformanceIssue.SeverityLevel.Medium
            });
        }
    }

    long EstimateTextureMemory(Texture texture)
    {
        // 简化的内存估算
        int width = texture.width;
        int height = texture.height;

        // 假设RGBA32格式（4字节/像素）
        long pixelCount = width * height;
        return pixelCount * 4;
    }

    void AnalyzeRaycastTargets()
    {
        int raycastTargetCount = 0;

        foreach (Canvas canvas in allCanvases)
        {
            Graphic[] graphics = canvas.GetComponentsInChildren<Graphic>(true);
            foreach (Graphic graphic in graphics)
            {
                if (graphic.raycastTarget)
                {
                    raycastTargetCount++;
                }
            }
        }

        if (raycastTargetCount > 50)
        {
            performanceIssues.Add(new PerformanceIssue
            {
                issueType = "射线检测目标过多",
                description = $"发现{raycastTargetCount}个射线检测目标",
                location = "多个UI元素",
                recommendation = "禁用不需要交互的元素的raycastTarget属性",
                severity = PerformanceIssue.SeverityLevel.Low
            });
        }
    }

    void GenerateReport()
    {
        reportBuilder.AppendLine("=== UI性能分析报告 ===");
        reportBuilder.AppendLine($"分析时间: {System.DateTime.Now}");
        reportBuilder.AppendLine($"Canvas数量: {allCanvases.Length}");
        reportBuilder.AppendLine($"发现问题: {performanceIssues.Count}个");
        reportBuilder.AppendLine();

        // 按严重程度分组
        var issuesBySeverity = new Dictionary<PerformanceIssue.SeverityLevel, List<PerformanceIssue>>();

        foreach (var level in System.Enum.GetValues(typeof(PerformanceIssue.SeverityLevel)))
        {
            issuesBySeverity[(PerformanceIssue.SeverityLevel)level] = new List<PerformanceIssue>();
        }

        foreach (PerformanceIssue issue in performanceIssues)
        {
            issuesBySeverity[issue.severity].Add(issue);
        }

        // 按严重程度从高到低输出
        foreach (var level in new[] {
            PerformanceIssue.SeverityLevel.Critical,
            PerformanceIssue.SeverityLevel.High,
            PerformanceIssue.SeverityLevel.Medium,
            PerformanceIssue.SeverityLevel.Low
        })
        {
            var issues = issuesBySeverity[level];
            if (issues.Count > 0)
            {
                reportBuilder.AppendLine($"【{level}级别问题 - {issues.Count}个】");

                foreach (PerformanceIssue issue in issues)
                {
                    reportBuilder.AppendLine($"• {issue.issueType}");
                    reportBuilder.AppendLine($"  描述: {issue.description}");
                    reportBuilder.AppendLine($"  位置: {issue.location}");
                    reportBuilder.AppendLine($"  建议: {issue.recommendation}");
                    reportBuilder.AppendLine();
                }
            }
        }

        // 优化建议摘要
        reportBuilder.AppendLine("=== 优化建议摘要 ===");

        if (performanceIssues.Count == 0)
        {
            reportBuilder.AppendLine("✓ UI性能良好，未发现明显问题");
        }
        else
        {
            reportBuilder.AppendLine("主要优化方向:");

            var recommendations = new HashSet<string>();
            foreach (PerformanceIssue issue in performanceIssues)
            {
                if (issue.severity >= PerformanceIssue.SeverityLevel.Medium)
                {
                    recommendations.Add(issue.recommendation);
                }
            }

            foreach (string rec in recommendations)
            {
                reportBuilder.AppendLine($"• {rec}");
            }
        }

        Debug.Log(reportBuilder.ToString());
    }

    void DisplayWarnings()
    {
        // 在游戏中显示关键警告
        foreach (PerformanceIssue issue in performanceIssues)
        {
            if (issue.severity >= PerformanceIssue.SeverityLevel.High)
            {
                Debug.LogWarning($"UI性能警告: {issue.issueType} - {issue.description}");
            }
        }
    }

    void TogglePerformanceDisplay()
    {
        // 切换游戏内性能显示
        showInGameDisplay = !showInGameDisplay;

        if (showInGameDisplay)
        {
            // 创建或显示性能显示UI
            Debug.Log("性能显示已开启");
        }
        else
        {
            // 隐藏性能显示UI
            Debug.Log("性能显示已关闭");
        }
    }

    // 公共方法
    public string GetPerformanceReport()
    {
        return reportBuilder.ToString();
    }

    public List<PerformanceIssue> GetPerformanceIssues()
    {
        return new List<PerformanceIssue>(performanceIssues);
    }

    public void SaveReportToFile(string filePath)
    {
        try
        {
            System.IO.File.WriteAllText(filePath, reportBuilder.ToString());

        }

        {
            Debug.LogError($"保存报告失败: {e.Message}");
        }
    }
```

```csharp
// Unity多平台UI适配系统
using UnityEngine;
using UnityEngine.UI;
using System.Collections.Generic;

public class MultiplatformUIAdapter : MonoBehaviour
{
    [System.Serializable]
    public class PlatformSettings
    {
        public RuntimePlatform platform;
        public float scaleFactor = 1f;
        public float safeAreaPadding = 0f;
        public Font overrideFont;
        public int fontSizeMultiplier = 100; // 百分比
        public float buttonMinSize = 44f; // 像素
        public bool enableHapticFeedback = true;
    }

    [Header("平台设置")]
    public PlatformSettings[] platformSettings;
    public PlatformSettings defaultSettings;

    [Header("适配组件")]
    public CanvasScaler canvasScaler;
    public RectTransform safeAreaGuide;
    public bool adaptOnStart = true;

    private Dictionary<RuntimePlatform, PlatformSettings> settingsDictionary;
    private PlatformSettings currentSettings;

    void Start()
    {
        InitializeAdapter();

        if (adaptOnStart)
        {
            AdaptToCurrentPlatform();
        }
    }

    void InitializeAdapter()
    {
        settingsDictionary = new Dictionary<RuntimePlatform, PlatformSettings>();

        foreach (PlatformSettings settings in platformSettings)
        {
            settingsDictionary[settings.platform] = settings;
        }

        // 设置默认值
        if (defaultSettings == null)
        {
            defaultSettings = new PlatformSettings
            {
                scaleFactor = 1f,
                safeAreaPadding = 0f,
                fontSizeMultiplier = 100,
                buttonMinSize = 44f,
                enableHapticFeedback = true
            };
        }
    }

    public void AdaptToCurrentPlatform()
    {
        RuntimePlatform currentPlatform = Application.platform;

        if (settingsDictionary.ContainsKey(currentPlatform))
        {
            currentSettings = settingsDictionary[currentPlatform];
        }
        else
        {
            currentSettings = defaultSettings;
            Debug.Log($"未找到{currentPlatform}的特定设置，使用默认设置");
        }

        ApplyPlatformSettings(currentSettings);

        Debug.Log($"UI已适配到平台: {currentPlatform}");
    }

    void ApplyPlatformSettings(PlatformSettings settings)
    {
        // 应用Canvas缩放
        if (canvasScaler != null)
        {
            canvasScaler.scaleFactor = settings.scaleFactor;
        }

        // 应用安全区域
        ApplySafeArea(settings.safeAreaPadding);

        // 应用字体设置
        ApplyFontSettings(settings);

        // 调整按钮大小
        AdjustButtonSizes(settings.buttonMinSize);

        // 配置触觉反馈
        ConfigureHapticFeedback(settings.enableHapticFeedback);
    }

    void ApplySafeArea(float padding)
    {
        if (safeAreaGuide == null) return;

        // 获取屏幕安全区域
        Rect safeArea = Screen.safeArea;

        // 转换为Canvas空间
        Canvas canvas = GetComponentInParent<Canvas>();
        if (canvas == null) return;

        RectTransform canvasRect = canvas.GetComponent<RectTransform>();

        // 计算安全区域在Canvas中的位置和大小
        Vector2 anchorMin = safeArea.position;
        Vector2 anchorMax = safeArea.position + safeArea.size;

        anchorMin.x /= Screen.width;
        anchorMin.y /= Screen.height;
        anchorMax.x /= Screen.width;
        anchorMax.y /= Screen.height;

        // 应用内边距
        anchorMin.x += padding / Screen.width;
        anchorMin.y += padding / Screen.height;
        anchorMax.x -= padding / Screen.width;
        anchorMax.y -= padding / Screen.height;

        safeAreaGuide.anchorMin = anchorMin;
        safeAreaGuide.anchorMax = anchorMax;

        // 将子UI元素限制在安全区域内
        RestrictChildrenToSafeArea();
    }

    void RestrictChildrenToSafeArea()
    {
        if (safeAreaGuide == null) return;

        foreach (Transform child in transform)
        {
            if (child == safeAreaGuide.transform) continue;

            RectTransform childRect = child.GetComponent<RectTransform>();
            if (childRect != null)
            {
                // 检查是否在安全区域外
                Vector3[] childCorners = new Vector3[4];
                Vector3[] safeAreaCorners = new Vector3[4];

                childRect.GetWorldCorners(childCorners);
                safeAreaGuide.GetWorldCorners(safeAreaCorners);

                // 简单检查：如果子元素完全在安全区域内，则不需要调整
                // 实际项目中可能需要更复杂的逻辑
            }
        }
    }

    void ApplyFontSettings(PlatformSettings settings)
    {
        if (settings.overrideFont != null)
        {
            // 更换所有文本字体
            Text[] allTexts = GetComponentsInChildren<Text>(true);
            foreach (Text text in allTexts)
            {
                text.font = settings.overrideFont;
            }
        }

        // 调整字体大小
        if (settings.fontSizeMultiplier != 100)
        {
            Text[] allTexts = GetComponentsInChildren<Text>(true);
            foreach (Text text in allTexts)
            {
                text.fontSize = Mathf.RoundToInt(text.fontSize * settings.fontSizeMultiplier / 100f);
            }
        }
    }

    void AdjustButtonSizes(float minSize)
    {
        Button[] allButtons = GetComponentsInChildren<Button>(true);

        foreach (Button button in allButtons)
        {
            RectTransform rectTransform = button.GetComponent<RectTransform>();
            if (rectTransform != null)
            {
                // 确保按钮不小于最小尺寸
                Vector2 size = rectTransform.sizeDelta;

                if (size.x < minSize) size.x = minSize;
                if (size.y < minSize) size.y = minSize;

                rectTransform.sizeDelta = size;
            }
        }
    }

    void ConfigureHapticFeedback(bool enable)
    {
        // 查找并配置所有触觉反馈组件
        HapticFeedbackSystem[] hapticSystems = FindObjectsOfType<HapticFeedbackSystem>();
        foreach (HapticFeedbackSystem system in hapticSystems)
        {
            system.SetHapticsEnabled(enable);
        }
    }

    // 动态适配分辨率变化
    void OnRectTransformDimensionsChange()
    {
        if (currentSettings != null)
        {
            ApplySafeArea(currentSettings.safeAreaPadding);
        }
    }

    // 公共方法
    public void SwitchToPlatform(RuntimePlatform platform)
    {
        if (settingsDictionary.ContainsKey(platform))
        {
            ApplyPlatformSettings(settingsDictionary[platform]);
            currentSettings = settingsDictionary[platform];
        }
        else
        {
            Debug.LogWarning($"未找到{platform}的平台设置");
        }
    }

    public PlatformSettings GetCurrentSettings()
    {
        return currentSettings;
    }

    // 编辑器工具
    #if UNITY_EDITOR
    [ContextMenu("测试手机适配")]
    void TestMobileAdaptation()
    {
        SwitchToPlatform(RuntimePlatform.Android);
    }

    [ContextMenu("测试PC适配")]
    void TestPCAdaptation()
    {
        SwitchToPlatform(RuntimePlatform.WindowsPlayer);
    }

    [UnityEditor.MenuItem("Tools/UI/创建平台适配器")]
    static void CreatePlatformAdapter()
    {
        GameObject adapter = new GameObject("Platform UI Adapter");
        adapter.AddComponent<MultiplatformUIAdapter>();

        // 添加Canvas Scaler
        Canvas canvas = adapter.AddComponent<Canvas>();
        canvas.renderMode = RenderMode.ScreenSpaceOverlay;

        CanvasScaler scaler = adapter.AddComponent<CanvasScaler>();
        scaler.uiScaleMode = CanvasScaler.ScaleMode.ScaleWithScreenSize;
        scaler.referenceResolution = new Vector2(1920, 1080);
        scaler.screenMatchMode = CanvasScaler.ScreenMatchMode.MatchWidthOrHeight;
        scaler.matchWidthOrHeight = 0.5f;

        // 创建安全区域引导
        GameObject safeArea = new GameObject("Safe Area Guide");
        safeArea.transform.SetParent(adapter.transform, false);
        safeArea.AddComponent<Image>().color = new Color(0, 1, 0, 0.1f);

        RectTransform safeAreaRT = safeArea.GetComponent<RectTransform>();
        safeAreaRT.anchorMin = new Vector2(0.1f, 0.1f);
        safeAreaRT.anchorMax = new Vector2(0.9f, 0.9f);

        MultiplatformUIAdapter adapterComponent = adapter.GetComponent<MultiplatformUIAdapter>();
        adapterComponent.canvasScaler = scaler;
        adapterComponent.safeAreaGuide = safeAreaRT;

        UnityEditor.Selection.activeGameObject = adapter;
    }
    #endif
}
```

## 总结

用户界面美术设计是游戏开发中连接玩家与游戏体验的关键桥梁。优秀的UI设计不仅需要视觉美感，更需要深刻理解用户需求、交互逻辑和技术实现。

**核心设计原则回顾**：

1. **清晰性优先**：信息传达永远比视觉华丽更重要
2. **一致性至上**：统一的视觉语言和交互模式创造专业体验
3. **响应性必备**：即时反馈让玩家感觉控制有力
4. **可访问性必须**：考虑所有玩家的需求，包括残障人士
5. **性能意识**：华丽但卡顿的UI不如简单但流畅的UI

**给独立开发者的实用建议**：

1. **从纸面开始**：先画草图，再做数字设计
2. **建立设计系统**：创建可重用的组件库和样式指南
3. **原型测试**：在投入制作前用简单原型验证设计
4. **关注核心功能**：先确保基础UI完美工作，再添加高级功能
5. **多平台思维**：从一开始就考虑不同设备的适配需求

**技术趋势关注**：
1. **AI辅助设计**：AI工具在布局建议、图标生成、可用性测试中的应用
2. **实时协作工具**：基于云的UI设计协作平台
3. **增强现实UI**：AR/VR设备的特殊UI需求和技术
4. **语音与手势交互**：非传统输入方式的UI设计

**最终思考**：
记住：**最好的UI是玩家注意不到的UI**。当UI设计得如此自然流畅，以至于玩家完全专注于游戏内容时，你就成功了。UI应该是一个透明的媒介，而不是障碍。
